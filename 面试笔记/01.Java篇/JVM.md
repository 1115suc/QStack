# JVM 面试

## 1. Java 的执行流程

- 编写源代码：写java文件，这是人能看懂的代码
- 编译成字节码：javac编译器把源代码编译成.class文件，里面是JVM能认的字节码指令
- 类加载：JVM启动后，类加载器把.class文件读进内存，经过验证、准备、解析、初始化这几个阶段
- 执行字节码：解释器逐条翻译字节码为机器码执行，同时JIT编译器监控哪些代码是热点
- IT编译优化：跑得多的热点代码被JIT编译成机器码缓存起来，下次直接跑机器码，不用再解释
- 垃圾回收：程序运行过程中，GC线程在后台回收不用的对象，释放堆内存
- 程序退出：main方法返回，JVM做清理工作后退出
