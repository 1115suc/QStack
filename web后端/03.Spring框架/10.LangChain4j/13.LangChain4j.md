# LangChain4j 

## 一、模型上下文协议 MCP

MCP（Model Context Protocol）是由 Anthropic 主导的开放协议，允许 LLM 通过标准化接口调用外部服务（类似 OpenAPI，但专为 AI 设计）。LangChain4j 从 `0.36` 版本开始支持 MCP。

### 1.1 MCP 依赖引入

```xml
<!-- MCP 客户端（Stdio 传输） -->
<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j-mcp</artifactId>
</dependency>

<!-- MCP HTTP/SSE 传输支持 -->
<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j-mcp-http</artifactId>
</dependency>
```

### 1.2 核心概念

| 概念 | 说明 |
|---|---|
| **MCP Server** | 提供工具/资源/提示词的外部服务进程 |
| **McpClient** | LangChain4j 侧的连接客户端 |
| **McpToolProvider** | 将 MCP Server 的工具包装为 LangChain4j 可用的工具 |
| **Stdio 传输** | 通过标准输入/输出与本地 MCP Server 进程通信 |
| **HTTP/SSE 传输** | 通过 HTTP + Server-Sent Events 与远程 MCP Server 通信 |

### 1.3 连接本地 MCP Server（Stdio 模式）

Stdio 模式适合连接本地运行的 MCP Server（如文件系统、数据库等）。

```java
@Configuration
public class McpConfig {

    /**
     * 连接本地文件系统 MCP Server（Node.js 进程）
     */
    @Bean
    public McpClient filesystemMcpClient() throws Exception {
        McpClient client = McpClient.builder()
                .transport(
                    StdioMcpTransport.builder()
                        // 启动命令：npx -y @modelcontextprotocol/server-filesystem /path/to/dir
                        .command(List.of(
                            "npx", "-y",
                            "@modelcontextprotocol/server-filesystem",
                            "/tmp/ai-workspace"
                        ))
                        .logEvents(true) // 记录 MCP 通信日志
                        .build()
                )
                .clientName("QAssistant-FileSystem")
                .clientVersion("1.0.0")
                .build();

        client.initialize(); // 建立连接，握手
        return client;
    }

    /**
     * 连接本地 SQLite MCP Server
     */
    @Bean
    public McpClient sqliteMcpClient() throws Exception {
        McpClient client = McpClient.builder()
                .transport(
                    StdioMcpTransport.builder()
                        .command(List.of(
                            "npx", "-y",
                            "@modelcontextprotocol/server-sqlite",
                            "--db-path", "/data/app.db"
                        ))
                        .build()
                )
                .build();

        client.initialize();
        return client;
    }
}
```

### 1.4 连接远程 MCP Server（HTTP/SSE 模式）

HTTP/SSE 适合连接部署在服务器上的 MCP Server（如企业内部服务、第三方 API 封装等）。

```java
@Bean
public McpClient remoteMcpClient() throws Exception {
    McpClient client = McpClient.builder()
            .transport(
                HttpMcpTransport.builder()
                    .sseUrl("https://your-mcp-server.com/sse") // SSE 端点
                    .timeout(Duration.ofSeconds(60))
                    .logRequests(true)
                    .logResponses(true)
                    // 如需认证
                    .customHeaders(Map.of(
                        "Authorization", "Bearer " + mcpServerToken
                    ))
                    .build()
            )
            .build();

    client.initialize();
    return client;
}
```

## 二、检索增强生成 RAG

RAG（Retrieval-Augmented Generation）将文档向量化存储，在用户提问时检索最相关的片段作为上下文，让模型基于知识库回答，显著减少幻觉。

### 2.1 RAG 核心架构

```
【Ingestion 阶段（数据入库）】
文档 → 分割（DocumentSplitter）→ 向量化（EmbeddingModel）→ 存储（EmbeddingStore）

【Retrieval 阶段（检索）】
用户问题 → 向量化 → 相似度搜索 → 返回 TopK 片段

【Generation 阶段（生成）】
系统提示词 + 检索结果 + 用户问题 → LLM → 最终回答
```

**Chroma 元数据过滤（按文件 ID 检索）：**

```java
@Service
@RequiredArgsConstructor
public class ChromaRagService {

    private final EmbeddingModel embeddingModel;
    private final EmbeddingStore<TextSegment> chromaEmbeddingStore;

    /**
     * 按 fileId 过滤检索（隔离不同文档的知识库）
     */
    public List<EmbeddingMatch<TextSegment>> searchByFileId(
            String query, Long fileId, int maxResults) {

        TextSegment querySegment = TextSegment.from(query);
        Embedding queryEmbedding = embeddingModel.embed(querySegment).content();

        // 构建 Metadata 过滤器
        Filter fileFilter = MetadataFilterBuilder.metadataKey("fileId")
                .isEqualTo(fileId.toString());

        EmbeddingSearchRequest request = EmbeddingSearchRequest.builder()
                .queryEmbedding(queryEmbedding)
                .maxResults(maxResults)
                .minScore(0.7)
                .filter(fileFilter)
                .build();

        return chromaEmbeddingStore.search(request).matches();
    }

    /**
     * 组合过滤（文件ID + 会话ID）
     */
    public List<EmbeddingMatch<TextSegment>> searchWithCombinedFilter(
            String query, Long fileId, String sessionId, int maxResults) {

        Embedding queryEmbedding = embeddingModel.embed(TextSegment.from(query)).content();

        Filter combinedFilter = new And(
                MetadataFilterBuilder.metadataKey("fileId").isEqualTo(fileId.toString()),
                MetadataFilterBuilder.metadataKey("sessionId").isEqualTo(sessionId)
        );

        EmbeddingSearchRequest request = EmbeddingSearchRequest.builder()
                .queryEmbedding(queryEmbedding)
                .maxResults(maxResults)
                .minScore(0.65)
                .filter(combinedFilter)
                .build();

        return chromaEmbeddingStore.search(request).matches();
    }
}
```

**Qdrant Collection 初始化工具类：**

```java
@Slf4j
@RequiredArgsConstructor
public class QdrantCollectionInitializer implements InitializingBean {

    private final String host;
    private final int port;
    private final String collectionName;
    private final int vectorDimension;

    @Override
    public void afterPropertiesSet() {
        try (ManagedChannel channel = ManagedChannelBuilder
                .forAddress(host, port)
                .usePlaintext()
                .build()) {

            CollectionsGrpc.CollectionsBlockingStub collectionsStub =
                    CollectionsGrpc.newBlockingStub(channel);

            // 检查 Collection 是否存在
            try {
                collectionsStub.get(
                        Collections.GetCollectionInfoRequest.newBuilder()
                                .setCollectionName(collectionName)
                                .build()
                );
                log.info("Qdrant Collection [{}] 已存在", collectionName);
            } catch (StatusRuntimeException e) {
                if (e.getStatus().getCode() == Status.Code.NOT_FOUND) {
                    // 创建 Collection
                    collectionsStub.create(
                            Collections.CreateCollection.newBuilder()
                                    .setCollectionName(collectionName)
                                    .setVectorsConfig(
                                        Collections.VectorsConfig.newBuilder()
                                            .setParams(
                                                Collections.VectorParams.newBuilder()
                                                    .setSize(vectorDimension)
                                                    .setDistance(Collections.Distance.Cosine)
                                                    .build()
                                            )
                                    )
                                    .build()
                    );
                    log.info("Qdrant Collection [{}] 创建成功，维度：{}", collectionName, vectorDimension);
                }
            }
        }
    }
}
```

**Qdrant Payload 过滤检索：**

```java
@Service
@RequiredArgsConstructor
public class QdrantRagService {

    private final EmbeddingModel embeddingModel;

    @Qualifier("qdrantStore")
    private final EmbeddingStore<TextSegment> qdrantStore;

    /**
     * 按 fileId 列表过滤（支持多文件检索）
     */
    public List<EmbeddingMatch<TextSegment>> searchByFileIds(
            String query, List<Long> fileIds, int maxResults) {

        Embedding queryEmbedding = embeddingModel.embed(TextSegment.from(query)).content();

        // Qdrant 支持 isIn 过滤（多值匹配）
        Filter fileFilter = MetadataFilterBuilder.metadataKey("fileId")
                .isIn(fileIds.stream().map(Object::toString).toList());

        EmbeddingSearchRequest request = EmbeddingSearchRequest.builder()
                .queryEmbedding(queryEmbedding)
                .maxResults(maxResults)
                .minScore(0.65)
                .filter(fileFilter)
                .build();

        return qdrantStore.search(request).matches();
    }

    /**
     * 包含字符串过滤（Qdrant 特有功能）
     * 例如：过滤 tags 字段包含某关键词的文档
     */
    public List<EmbeddingMatch<TextSegment>> searchWithContainsFilter(
            String query, String tagKeyword) {

        Embedding queryEmbedding = embeddingModel.embed(TextSegment.from(query)).content();

        // ContainsString 是 Qdrant 特有的过滤器
        Filter tagFilter = MetadataFilterBuilder.metadataKey("tags")
                .containsString(tagKeyword);

        EmbeddingSearchRequest request = EmbeddingSearchRequest.builder()
                .queryEmbedding(queryEmbedding)
                .maxResults(5)
                .filter(tagFilter)
                .build();

        return qdrantStore.search(request).matches();
    }
}
```

### 2.5 文档入库（Ingestion Pipeline）

```java
@Service
@RequiredArgsConstructor
@Slf4j
public class DocumentIngestionService {

    private final EmbeddingModel embeddingModel;

    @Qualifier("qdrantStore")
    private final EmbeddingStore<TextSegment> qdrantStore;

    // 也可以同时写入两个 Store
    @Qualifier("chromaStore")
    private final EmbeddingStore<TextSegment> chromaStore;

    /**
     * 通用文档入库（支持 PDF、TXT、MD 等格式）
     */
    public void ingestDocument(String filePath, Long fileId, String sessionId) {
        // 1. 加载文档
        Document document = loadDocument(filePath);

        // 2. 分割文档
        DocumentSplitter splitter = DocumentSplitters.recursive(
                512,  // 每段最大 Token 数
                64    // 段落重叠 Token 数（保证上下文连贯）
        );
        List<TextSegment> segments = splitter.split(document);

        // 3. 为每段注入 Metadata（用于后续过滤）
        List<TextSegment> segmentsWithMeta = segments.stream()
                .map(segment -> TextSegment.from(
                        segment.text(),
                        Metadata.from(Map.of(
                                "fileId", fileId.toString(),
                                "sessionId", sessionId,
                                "fileName", Paths.get(filePath).getFileName().toString(),
                                "source", filePath
                        ))
                ))
                .toList();

        // 4. 批量向量化并存储
        List<Embedding> embeddings = embeddingModel.embedAll(segmentsWithMeta).content();

        qdrantStore.addAll(embeddings, segmentsWithMeta);
        log.info("文档入库完成：fileId={}, 段落数={}", fileId, segments.size());
    }

    /**
     * 从 MinIO/S3 加载文档
     */
    public void ingestFromMinIO(String bucketName, String objectKey, Long fileId) {
        // 先下载到临时目录
        String tempPath = "/tmp/" + objectKey;
        minioClient.downloadFile(bucketName, objectKey, tempPath);
        ingestDocument(tempPath, fileId, "global");
        Files.deleteIfExists(Path.of(tempPath));
    }

    private Document loadDocument(String filePath) {
        if (filePath.endsWith(".pdf")) {
            return FileSystemDocumentLoader.loadDocument(
                    filePath, new ApachePdfBoxDocumentParser()
            );
        } else {
            return FileSystemDocumentLoader.loadDocument(filePath);
        }
    }
}
```

```java
// 方式二：编程式 RAG Pipeline（更灵活）
@Service
@RequiredArgsConstructor
public class CustomRagService {

    private final ChatLanguageModel chatLanguageModel;
    private final EmbeddingModel embeddingModel;
    private final EmbeddingStore<TextSegment> qdrantStore;

    public String ragChat(String sessionId, String question, Long fileId) {
        // 1. 检索相关文档
        Embedding questionEmbedding = embeddingModel
                .embed(TextSegment.from(question)).content();

        Filter fileFilter = MetadataFilterBuilder.metadataKey("fileId")
                .isEqualTo(fileId.toString());

        List<EmbeddingMatch<TextSegment>> matches = qdrantStore.search(
                EmbeddingSearchRequest.builder()
                        .queryEmbedding(questionEmbedding)
                        .maxResults(5)
                        .minScore(0.7)
                        .filter(fileFilter)
                        .build()
        ).matches();

        if (matches.isEmpty()) {
            return "抱歉，在知识库中未找到与您问题相关的内容。";
        }

        // 2. 构建上下文
        String context = matches.stream()
                .map(match -> "【参考资料】\n" + match.embedded().text())
                .collect(Collectors.joining("\n\n"));

        // 3. 构建 Prompt 并调用 LLM
        String prompt = String.format("""
                请基于以下参考资料回答用户的问题：
                
                %s
                
                用户问题：%s
                
                请直接回答，如需引用资料请注明来源。
                """, context, question);

        return chatLanguageModel.generate(
                List.of(
                    SystemMessage.from("你是一个专业的知识库问答助手。"),
                    UserMessage.from(prompt)
                )
        ).content().text();
    }
}
```

## 三、Skills（Dynamic Tools）

Skills 是 LangChain4j 中对工具调用能力的高层封装，允许将**可复用的业务能力**模块化，动态挂载到不同的 AI Service 中。

### 3.1 Skills 概念与 @Tool 的区别

| 特性 | `@Tool`（静态） | Skills（动态） |
|---|---|---|
| 注册方式 | 编译时注解 | 运行时动态注入 |
| 复用性 | 需重复声明 | 高度可复用 |
| 可组合性 | 有限 | 任意组合 |
| 适用场景 | 固定工具集 | 多租户/动态场景 |

### 3.2 定义可复用 Skill

```java
/**
 * 考试题目生成 Skill
 */
@Component
public class ExamGenerationSkill {

    @Tool("根据知识点生成单选题")
    public String generateMultipleChoice(
            @P("知识点描述") String knowledge,
            @P("难度：简单/中等/困难") String difficulty,
            @P("题目数量") int count) {
        // 实际调用 AI 或规则引擎生成题目
        return examService.generateMultipleChoice(knowledge, difficulty, count);
    }
```

## 四、AI Agents（自主代理）

Agent 是 LangChain4j 中最强大的模式，模型自主决策调用哪些工具、调用多少次，直到任务完成。

### 4.1 ReAct Agent（推理 + 行动）

```java
@AiService(
    wiringMode = AiServiceWiringMode.EXPLICIT,
    chatLanguageModel = "primaryChatModel",
    chatMemoryProvider = "chatMemoryProvider",
    tools = {WeatherTool.class, CalculatorTool.class, DatabaseQueryTool.class}
)
public interface ReactAgent {

    @SystemMessage("""
            你是一个能够自主完成复杂任务的 AI 代理。
            分析用户的请求，拆解为子任务，逐步调用工具完成。
            每次工具调用后，基于结果决定下一步行动。
            直到任务完全完成，再给出最终答案。
            """)
    String execute(@MemoryId String agentId, @UserMessage String task);
}
```

### 4.2 并行工具调用（提升效率）

```java
@Service
@RequiredArgsConstructor
public class ParallelToolCallService {

    private final ChatLanguageModel chatLanguageModel;
    private final WeatherTool weatherTool;
    private final CalculatorTool calculatorTool;

    /**
     * 使用 CompletableFuture 并行执行多个工具
     */
    public Map<String, String> parallelToolExecution(List<String> cities) {
        List<CompletableFuture<Map.Entry<String, String>>> futures = cities.stream()
                .map(city -> CompletableFuture.supplyAsync(() ->
                        Map.entry(city, weatherTool.getWeather(city))
                ))
                .toList();

        return futures.stream()
                .map(CompletableFuture::join)
                .collect(Collectors.toMap(Map.Entry::getKey, Map.Entry::getValue));
    }
}
```

### 6.3 Token 成本控制

```java
@Aspect
@Component
@Slf4j
public class TokenCostAspect {

    private final AtomicLong totalInputTokens = new AtomicLong(0);
    private final AtomicLong totalOutputTokens = new AtomicLong(0);

    @Around("execution(* dev.langchain4j.model.chat.ChatLanguageModel.generate(..))")
    public Object trackTokenUsage(ProceedingJoinPoint joinPoint) throws Throwable {
        Object result = joinPoint.proceed();
        if (result instanceof Response<?> response && response.tokenUsage() != null) {
            long inputTokens = response.tokenUsage().inputTokenCount();
            long outputTokens = response.tokenUsage().outputTokenCount();
            totalInputTokens.addAndGet(inputTokens);
            totalOutputTokens.addAndGet(outputTokens);
            log.debug("[Token 统计] 本次 in={}, out={}，累计 in={}, out={}",
                    inputTokens, outputTokens,
                    totalInputTokens.get(), totalOutputTokens.get());
        }
        return result;
    }
}
```
