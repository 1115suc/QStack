# Spring AI Alibaba 基础使用指南

## 1. Spring AI Alibaba 简介

Spring AI Alibaba 是基于 Spring AI 构建的阿里云开源框架，专为阿里云通义大模型（DashScope）系列提供最佳 Java 实践。

### 核心定位

```
Spring AI（标准接口层）
       ↑ 实现
Spring AI Alibaba（阿里云扩展层）
       ↓ 接入
DashScope / 阿里云百炼平台
       ↓ 提供
通义千问（Qwen）/ 通义万相（Wanx）/ ...
```

### 与 Spring AI 的关系

| 维度 | Spring AI | Spring AI Alibaba |
|---|---|---|
| 定位 | 通用 AI 框架抽象 | 阿里云通义模型最佳实践 |
| 核心接口 | `ChatClient` / `ChatModel` | 继承，额外提供 `DashScopeChatModel` |
| 模型选项 | `ChatOptions` | `DashScopeChatOptions`（扩展） |
| 特有能力 | — | 多模态（视觉/音频/视频）、通义 Embedding、StateGraph 工作流 |
| Embedding | 通用接口 | `DashScopeEmbeddingModel`（text-embedding-v3/v4） |
| MCP | 标准 MCP Client | 额外支持 Nacos MCP Registry |

### 支持的通义模型清单

| 分类 | 模型名称 | 说明 |
|---|---|---|
| 文本对话 | `qwen-turbo` / `qwen-plus` / `qwen-max` | 通用对话，性能由弱到强 |
| 文本对话 | `qwen3` / `qwen3-plus` / `qwen3-max` | Qwen3 系列，支持思考模式 |
| 视觉多模态 | `qwen-vl-plus` / `qwen-vl-max` | 图片理解 |
| 视觉多模态 | `qwen-vl-max-latest` | 最新视觉模型，支持高分辨率 |
| 音频多模态 | `qwen-audio-chat` | 音频理解 |
| Embedding | `text-embedding-v3` / `text-embedding-v4` | 文本向量化，支持自定义维度 |
| 图像生成 | `wanx-v1` / `wanx2.1-t2i-turbo` | 通义万相文生图 |

## 2. 项目快速引入

### 2.1 获取 API Key

1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com)
2. 注册/登录阿里云账号
3. 进入「API-KEY 管理」页面，创建 API Key
4. 设置环境变量：

```bash
# Linux / macOS
export AI_DASHSCOPE_API_KEY=sk-xxxxxxxxxxxxxxxx

# Windows CMD
set AI_DASHSCOPE_API_KEY=sk-xxxxxxxxxxxxxxxx

# Windows PowerShell
$env:AI_DASHSCOPE_API_KEY="sk-xxxxxxxxxxxxxxxx"
```

## 3. 模型配置详解

### 3.1 基础配置（application.yml）

```yaml
spring:
  ai:
    dashscope:
      # API Key（推荐使用环境变量）
      api-key: ${AI_DASHSCOPE_API_KEY}
      chat:
        options:
          # 默认对话模型
          model: qwen-plus
          # 温度参数（0~2，越高越随机，默认 0.7）
          temperature: 0.7
          # 最大输出 Token 数
          max-tokens: 2048
          # Top-P 核采样（0~1，默认 0.9）
          top-p: 0.9
          # 重复惩罚（>1 减少重复，<1 增加重复）
          repetition-penalty: 1.0
          # 是否开启网络搜索增强（部分模型支持）
          enable-search: false
      embedding:
        options:
          # Embedding 模型
          model: text-embedding-v3
```

### 3.2 可用配置参数说明

| 配置项 | 说明 | 推荐值 |
|---|---|---|
| `spring.ai.dashscope.api-key` | DashScope API Key | 从环境变量读取 |
| `chat.options.model` | 对话模型名称 | `qwen-plus`（均衡）/ `qwen-max`（高质量） |
| `chat.options.temperature` | 创意程度（0~2） | 对话：0.7，代码：0.2，创意写作：1.0 |
| `chat.options.max-tokens` | 最大输出 Token | 2048~8192 |
| `chat.options.top-p` | 核采样参数 | 0.9 |
| `chat.options.enable-search` | 是否启用联网搜索 | `false`（按需开启） |
| `embedding.options.model` | Embedding 模型 | `text-embedding-v3` |

## 4. 基础聊天使用

Spring AI Alibaba 完全兼容 Spring AI 的 `ChatClient` Fluent API，同时支持底层 `DashScopeChatModel`。

### 4.1 使用 ChatClient（推荐方式）

```java
@RestController
@RequestMapping("/ai/chat")
public class BasicChatController {

    private final ChatClient chatClient;

    // 注入 @Primary 的 ChatClient Bean
    public BasicChatController(ChatClient.Builder chatClientBuilder) {
        this.chatClient = chatClientBuilder
                .defaultSystem("你是一个博学的智能聊天助手，请根据用户提问回答！")
                .build();
    }

    /**
     * 最简单的单次问答
     * GET /ai/chat/simple?question=你好，介绍一下通义千问
     */
    @GetMapping("/simple")
    public String simpleChat(@RequestParam String question) {
        return chatClient.prompt()
                .user(question)
                .call()
                .content();
    }

    /**
     * 带自定义系统消息的问答
     * POST /ai/chat/custom
     */
    @PostMapping("/custom")
    public String customChat(@RequestBody ChatRequest request) {
        return chatClient.prompt()
                .system(request.systemPrompt())
                .user(request.userMessage())
                .call()
                .content();
    }

    /**
     * 携带动态模型参数的对话（按请求切换模型）
     */
    @PostMapping("/with-options")
    public String chatWithOptions(@RequestBody ChatWithOptionsRequest request) {
        return chatClient.prompt()
                .user(request.question())
                .options(DashScopeChatOptions.builder()
                        .withModel(request.model())          // 动态指定模型
                        .withTemperature(request.temperature())
                        .withMaxTokens(request.maxTokens())
                        .build())
                .call()
                .content();
    }

    /**
     * 返回完整 ChatResponse（含 Token 用量、停止原因等元数据）
     */
    @GetMapping("/full")
    public ChatResponseVO fullChat(@RequestParam String question) {
        ChatResponse response = chatClient.prompt()
                .user(question)
                .call()
                .chatResponse();

        // 获取 Token 用量
        Usage usage = response.getMetadata().getUsage();

        return new ChatResponseVO(
                response.getResult().getOutput().getText(),
                usage.getPromptTokens(),
                usage.getGenerationTokens(),
                usage.getTotalTokens()
        );
    }

    record ChatRequest(String systemPrompt, String userMessage) {}

    record ChatWithOptionsRequest(String question, String model,
                                   Double temperature, Integer maxTokens) {}

    record ChatResponseVO(String content, Long promptTokens,
                           Long completionTokens, Long totalTokens) {}
}
```

### 4.2 使用底层 DashScopeChatModel

```java
@Service
public class DashScopeChatService {

    private final DashScopeChatModel chatModel;

    public DashScopeChatService(DashScopeChatModel chatModel) {
        this.chatModel = chatModel;
    }

    /**
     * 底层 API：手动构建 Prompt
     */
    public String chat(String systemContent, String userContent) {
        List<Message> messages = List.of(
                new SystemMessage(systemContent),
                new UserMessage(userContent)
        );

        Prompt prompt = new Prompt(messages,
                DashScopeChatOptions.builder()
                        .withModel("qwen-plus")
                        .withTemperature(0.7)
                        .build());

        ChatResponse response = chatModel.call(prompt);
        return response.getResult().getOutput().getText();
    }

    /**
     * 使用 PromptTemplate（参数化提示词）
     */
    public String chatWithTemplate(String language, String topic) {
        PromptTemplate template = new PromptTemplate("""
                请用{language}写一段关于{topic}的简短介绍，
                控制在200字以内，语言通俗易懂。
                """);

        Prompt prompt = template.create(Map.of(
                "language", language,
                "topic", topic
        ));

        return chatModel.call(prompt).getResult().getOutput().getText();
    }

    /**
     * 开启联网搜索（qwen-plus/qwen-max 支持）
     */
    public String chatWithSearch(String question) {
        Prompt prompt = new Prompt(
                new UserMessage(question),
                DashScopeChatOptions.builder()
                        .withModel("qwen-plus")
                        .withEnableSearch(true)   // 开启联网搜索
                        .build()
        );

        return chatModel.call(prompt).getResult().getOutput().getText();
    }
}
```

### 4.3 多轮对话（手动维护上下文）

```java
@Service
public class ManualMultiTurnService {

    private final ChatClient chatClient;
    // 简单内存存储（生产环境使用 ChatMemory，见进阶篇）
    private final ConcurrentHashMap<String, List<Message>> sessions = new ConcurrentHashMap<>();

    public ManualMultiTurnService(ChatClient.Builder builder) {
        this.chatClient = builder
                .defaultSystem("你是一个专业助手，保持上下文连续性进行对话。")
                .build();
    }

    public String chat(String sessionId, String userInput) {
        List<Message> history = sessions.computeIfAbsent(sessionId,
                k -> new ArrayList<>(List.of(
                        new SystemMessage("你是一个专业助手，保持上下文连续性进行对话。")
                ))
        );

        history.add(new UserMessage(userInput));

        ChatResponse response = chatClient.prompt()
                .messages(history)
                .call()
                .chatResponse();

        String reply = response.getResult().getOutput().getText();
        history.add(new AssistantMessage(reply));

        return reply;
    }

    public void clearSession(String sessionId) {
        sessions.remove(sessionId);
    }
}
```

## 5. 流式输出

### 5.1 基础流式输出（SSE）

```java
@RestController
@RequestMapping("/ai/stream")
public class StreamChatController {

    private final ChatClient chatClient;

    public StreamChatController(ChatClient.Builder builder) {
        this.chatClient = builder
                .defaultSystem("你是一个专业助手，请详细回答用户问题。")
                .build();
    }

    /**
     * SSE 流式输出（前端用 EventSource 接收）
     * GET /ai/stream/sse?question=介绍一下通义千问的发展历程
     */
    @GetMapping(value = "/sse", produces = MediaType.TEXT_EVENT_STREAM_VALUE)
    public Flux<String> streamChat(@RequestParam String question) {
        return chatClient.prompt()
                .user(question)
                .stream()
                .content();    // 返回 Flux<String>，每元素是一个 Token 片段
    }

    /**
     * 返回完整 ChatResponse 流（含元数据）
     */
    @GetMapping(value = "/full-stream", produces = MediaType.TEXT_EVENT_STREAM_VALUE)
    public Flux<ChatResponse> fullStream(@RequestParam String question) {
        return chatClient.prompt()
                .user(question)
                .stream()
                .chatResponse();
    }

    /**
     * 聚合流式结果为完整字符串
     */
    @GetMapping("/collect")
    public Mono<String> streamAndCollect(@RequestParam String question) {
        return chatClient.prompt()
                .user(question)
                .stream()
                .content()
                .collectList()
                .map(tokens -> String.join("", tokens));
    }

    /**
     * 带系统消息的流式输出（POST）
     */
    @PostMapping(value = "/chat", produces = MediaType.TEXT_EVENT_STREAM_VALUE)
    public Flux<String> streamWithSystem(@RequestBody StreamRequest request) {
        return chatClient.prompt()
                .system(request.systemPrompt())
                .user(request.question())
                .options(DashScopeChatOptions.builder()
                        .withModel("qwen-plus")
                        .withTemperature(0.7)
                        .build())
                .stream()
                .content();
    }

    record StreamRequest(String systemPrompt, String question) {}
}
```

                new UserMessage(question),
                DashScopeChatOptions.builder()

        return chatModel.stream(prompt);

    public Flux<String> streamText(String question) {

                .map(resp -> resp.getResult().getOutput().getText())
                .filter(text -> text != null && !text.isEmpty());
    }
}

## 6. 多模态模型使用

通义千问 VL 系列（`qwen-vl-*`）支持图片、视频、音频的多模态理解。

> **⚠️ 关键注意事项：**
> - 使用多模态时必须通过 `DashScopeApiConstants.MESSAGE_FORMAT` 元数据指定媒体类型
> - 必须在 `DashScopeChatOptions` 中开启 `withMultiModel(true)`
> - 必须使用支持视觉的模型：`qwen-vl-plus` / `qwen-vl-max` / `qwen-vl-max-latest`

### 6.1 图片理解——URL 方式

```java
@Service
public class MultiModalService {

    private final DashScopeChatModel chatModel;

    public MultiModalService(DashScopeChatModel chatModel) {
        this.chatModel = chatModel;
    }

    /**
     * 通过图片 URL 进行图像理解
     */
    public String analyzeImageByUrl(String question, String imageUrl) throws Exception {
        // 1. 构建媒体列表
        List<Media> mediaList = List.of(
                new Media(MimeTypeUtils.IMAGE_JPEG, new URI(imageUrl))
        );

        // 2. 构建 UserMessage（文本 + 图片）
        UserMessage userMessage = UserMessage.builder()
                .text(question)
                .media(mediaList)
                .build();

        // 3. ⚠️ 关键：设置消息格式为 IMAGE（否则模型当纯文本处理）
        userMessage.getMetadata().put(
                DashScopeApiConstants.MESSAGE_FORMAT,
                MessageFormat.IMAGE
        );

        // 4. 构建 Prompt（必须指定视觉模型并开启多模态）
        Prompt prompt = new Prompt(
                userMessage,
                DashScopeChatOptions.builder()
                        .withModel("qwen-vl-max-latest")   // 视觉模型
                        .withMultiModel(true)               // ⚠️ 必须开启
                        .withVlHighResolutionImages(true)  // 高分辨率支持
                        .withTemperature(0.7)
                        .build()
        );

        return chatModel.call(prompt).getResult().getOutput().getText();
    }
}
```

### 6.2 图片理解——文件上传（Base64）

```java
/**
 * 通过上传文件进行图像理解
 */
public String analyzeUploadedImage(MultipartFile file, String question) throws Exception {
    byte[] imageBytes = file.getBytes();
    String base64 = Base64.getEncoder().encodeToString(imageBytes);
    MimeType mimeType = MimeTypeUtils.parseMimeType(
            Objects.requireNonNull(file.getContentType()));

    // 使用 ByteArrayResource 封装 Base64 数据
    List<Media> mediaList = List.of(
            new Media(mimeType, new ByteArrayResource(imageBytes))
    );

    UserMessage userMessage = UserMessage.builder()
            .text(question)
            .media(mediaList)
            .build();

    userMessage.getMetadata().put(
            DashScopeApiConstants.MESSAGE_FORMAT,
            MessageFormat.IMAGE
    );

    Prompt prompt = new Prompt(
            userMessage,
            DashScopeChatOptions.builder()
                    .withModel("qwen-vl-plus")
                    .withMultiModel(true)
                    .build()
    );

    return chatModel.call(prompt).getResult().getOutput().getText();
}
```

## 7. 结构化输出

Spring AI Alibaba 继承 Spring AI 的结构化输出能力，支持将模型响应自动解析为 Java 对象。

### 7.1 使用 `.entity()` 直接返回 POJO

```java
@Service
public class StructuredOutputService {

    private final ChatClient chatClient;

    public StructuredOutputService(ChatClient.Builder builder) {
        this.chatClient = builder.build();
    }

    // 定义输出结构
    record TravelRecommendation(
            String destination,
            String bestSeason,
            List<String> mustVisitSpots,
            String estimatedBudget,
            String transportAdvice
    ) {}

    /**
     * 获取结构化旅游推荐
     */
    public TravelRecommendation getTravelRecommendation(String query) {
        return chatClient.prompt()
                .system("""
                        你是一名专业旅游顾问。
                        请为用户推荐旅游目的地，返回结构化的旅游信息。
                        """)
                .user(query)
                .call()
                .entity(TravelRecommendation.class);   // 自动解析为 POJO
    }

    /**
     * 返回对象列表
     */
    public List<TravelRecommendation> getMultipleRecommendations(String query) {
        return chatClient.prompt()
                .user("推荐3个适合" + query + "的旅游目的地，每个目的地单独列出详细信息。")
                .call()
                .entity(new ParameterizedTypeReference<List<TravelRecommendation>>() {});
    }
}
```

### 7.2 手动使用 BeanOutputConverter

```java
@Service
public class ManualStructuredOutputService {

    private final ChatClient chatClient;

    public ManualStructuredOutputService(ChatClient.Builder builder) {
        this.chatClient = builder.build();
    }

    record RecipeInfo(
            String name,
            List<String> ingredients,
            List<String> steps,
            int cookingTimeMinutes,
            int servings
    ) {}

    public RecipeInfo getRecipe(String dish) {
        BeanOutputConverter<RecipeInfo> converter =
                new BeanOutputConverter<>(RecipeInfo.class);

        // 获取格式说明（JSON Schema）
        String formatInstructions = converter.getFormat();

        String response = chatClient.prompt()
                .system("你是一名专业厨师，请提供详细的菜谱信息。")
                .user("""
                        请提供%s的完整菜谱。
                        
                        %s
                        """.formatted(dish, formatInstructions))
                .call()
                .content();

        return converter.convert(response);
    }
}
```

## 8. Prompt 模板

### 8.1 基础 PromptTemplate

```java
@Service
public class PromptTemplateService {

    private final ChatClient chatClient;

    public PromptTemplateService(ChatClient.Builder builder) {
        this.chatClient = builder.build();
    }

    /**
     * 使用 PromptTemplate 生成参数化提示词
     */
    public String generateReport(String topic, String audience, String style) {
        PromptTemplate template = new PromptTemplate("""
                请为{audience}写一篇关于{topic}的{style}报告。
                要求：
                - 语言{style}
                - 包含数据支撑
                - 字数 500-800 字
                """);

        Prompt prompt = template.create(Map.of(
                "topic", topic,
                "audience", audience,
                "style", style
        ));

        return chatClient.prompt()
                .user(prompt.getContents())
                .call()
                .content();
    }

    /**
     * 从 classpath 加载 Prompt 模板文件
     * src/main/resources/prompts/analysis.st
     */
    public String analyzeFromFile(String data, String dimension) {
        PromptTemplate template = new PromptTemplate(
                new ClassPathResource("prompts/analysis.st")
        );

        return chatClient.prompt()
                .user(template.render(Map.of("data", data, "dimension", dimension)))
                .call()
                .content();
    }
}
```

## 9. 常见问题与注意事项

### 9.1 依赖仓库未配置导致构建失败

**现象：** `Could not find artifact com.alibaba.cloud.ai:spring-ai-alibaba-starter`

**解决：** 在 `pom.xml` 中添加 Spring Milestones 仓库（见 2.2 节）。

### 9.2 多模态调用无效（模型把图片当文本处理）

**原因：** 未设置 `MESSAGE_FORMAT` 元数据，或未开启 `withMultiModel(true)`

**解决：**
```java
// 必须同时设置这两项
userMessage.getMetadata().put(DashScopeApiConstants.MESSAGE_FORMAT, MessageFormat.IMAGE);
DashScopeChatOptions.builder().withMultiModel(true).build();
```

### 9.3 API Key 安全管理

```yaml
# application.yml：使用环境变量，禁止硬编码
spring:
  ai:
    dashscope:
      api-key: ${AI_DASHSCOPE_API_KEY}  # 引用环境变量
```

### 9.4 Token 超限处理

通义千问各模型 Context Window 不同，建议：
- `qwen-turbo`：8K Token，适合短对话
- `qwen-plus`：128K Token，适合长文档
- `qwen-max`：128K Token，高质量长上下文

```yaml
spring:
  ai:
    dashscope:
      chat:
        options:
          max-tokens: 4096   # 根据模型 Context 合理设置
```

### 9.5 并发超限（429 错误）

```java
@Configuration
public class RetryConfig {

    @Bean
    public RetryTemplate aiRetryTemplate() {
        return RetryTemplate.builder()
                .maxAttempts(3)
                .exponentialBackoff(1000, 2, 10000)
                .retryOn(HttpClientErrorException.class)
                .build();
    }
}
```
