# JVM 指令手册

> JVM 指令是 Java 虚拟机执行字节码的基本操作单元，按功能可分为多个类别。以下是指令的完整分类及说明。

## 一、栈和局部变量操作

### 1.1 常量入栈指令

将常量值压入操作数栈顶。

| 指令 | 说明 |
|------|------|
| `aconst_null` | 将 null 对象引用压入栈 |
| `iconst_m1` | 将 int 类型常量 -1 压入栈 |
| `iconst_0` ~ `iconst_5` | 将 int 类型常量 0~5 压入栈 |
| `lconst_0`, `lconst_1` | 将 long 类型常量 0 或 1 压入栈 |
| `fconst_0`, `fconst_1` | 将 float 类型常量 0 或 1 压入栈 |
| `dconst_0`, `dconst_1` | 将 double 类型常量 0 或 1 压入栈 |
| `bipush` | 将 8 位带符号整数 (byte) 压入栈 |
| `sipush` | 将 16 位带符号整数 (short) 压入栈 |
| `ldc` | 把常量池中的项压入栈 |
| `ldc_w` | 把常量池中的项压入栈（使用宽索引） |
| `ldc2_w` | 把常量池中 long 或 double 类型的项压入栈（使用宽索引） |

### 1.2 从局部变量装载值到栈

从局部变量表加载值到操作数栈。

| 指令 | 说明 |
|------|------|
| `iload`, `iload_<n>` | 从局部变量装载 int 类型值 (0-3) |
| `lload`, `lload_<n>` | 从局部变量装载 long 类型值 (0-3) |
| `fload`, `fload_<n>` | 从局部变量装载 float 类型值 (0-3) |
| `dload`, `dload_<n>` | 从局部变量装载 double 类型值 (0-3) |
| `aload`, `aload_<n>` | 从局部变量装载引用类型值 (0-3) |
| `iaload` | 从 int 数组中装载值 |
| `laload` | 从 long 数组中装载值 |
| `faload` | 从 float 数组中装载值 |
| `daload` | 从 double 数组中装载值 |
| `aaload` | 从引用类型数组中装载值 |
| `baload` | 从 byte/boolean 数组中装载值 |
| `caload` | 从 char 数组中装载值 |
| `saload` | 从 short 数组中装载值 |

### 1.3 从栈存储值到局部变量

将操作数栈顶的值存入局部变量表。

| 指令 | 说明 |
|------|------|
| `istore`, `istore_<n>` | 将 int 类型值存入局部变量 (0-3) |
| `lstore`, `lstore_<n>` | 将 long 类型值存入局部变量 (0-3) |
| `fstore`, `fstore_<n>` | 将 float 类型值存入局部变量 (0-3) |
| `dstore`, `dstore_<n>` | 将 double 类型值存入局部变量 (0-3) |
| `astore`, `astore_<n>` | 将引用类型值存入局部变量 (0-3) |
| `iastore` | 将 int 类型值存入数组 |
| `lastore` | 将 long 类型值存入数组 |
| `fastore` | 将 float 类型值存入数组 |
| `dastore` | 将 double 类型值存入数组 |
| `aastore` | 将引用类型值存入数组 |
| `bastore` | 将 byte/boolean 类型值存入数组 |
| `castore` | 将 char 类型值存入数组 |
| `sastore` | 将 short 类型值存入数组 |

### 1.4 wide 指令

| 指令 | 说明 |
|------|------|
| `wide` | 使用附加字节扩展局部变量索引（用于索引大于 255 的情况） |

### 1.5 通用栈操作

无类型的栈操作指令。

| 指令 | 说明 |
|------|------|
| `nop` | 不做任何操作（占位符） |
| `pop` | 弹出栈顶一个字长（4 字节）的内容 |
| `pop2` | 弹出栈顶两个字长的内容 |
| `dup` | 复制栈顶一个字长的内容 |
| `dup_x1` | 复制栈顶字长，插入到顶部 2 个值之下 |
| `dup_x2` | 复制栈顶字长，插入到顶部 3 个值之下 |
| `dup2` | 复制栈顶两个字长的内容 |
| `dup2_x1` | 复制栈顶两字长，插入到顶部 3 个值之下 |
| `dup2_x2` | 复制栈顶两字长，插入到顶部 4 个值之下 |
| `swap` | 交换栈顶两个字长的内容 |

## 二、类型转换指令

数值类型之间的转换。

### 2.1 拓宽数值转换

| 指令 | 说明 |
|------|------|
| `i2l` | int → long |
| `i2f` | int → float |
| `i2d` | int → double |
| `l2f` | long → float |
| `l2d` | long → double |
| `f2d` | float → double |

### 2.2 缩窄数值转换

| 指令 | 说明 |
|------|------|
| `l2i` | long → int |
| `f2i` | float → int |
| `f2l` | float → long |
| `d2i` | double → int |
| `d2l` | double → long |
| `d2f` | double → float |
| `i2b` | int → byte |
| `i2c` | int → char |
| `i2s` | int → short |

## 五、对象与数组操作

### 5.1 对象操作

| 指令 | 说明 |
|------|------|
| `new` | 创建新对象实例 |
| `getfield` | 获取对象的实例字段值 |
| `putfield` | 设置对象的实例字段值 |
| `getstatic` | 获取类的静态字段值 |
| `putstatic` | 设置类的静态字段值 |
| `instanceof` | 判断对象是否为指定类型（是则压入 1，否则 0） |
| `checkcast` | 检查对象类型，失败则抛出 ClassCastException |

### 5.2 数组操作

| 指令 | 说明 |
|------|------|
| `newarray` | 创建基本数据类型数组 |
| `anewarray` | 创建引用类型数组 |
| `multianewarray` | 创建多维数组 |
| `arraylength` | 获取数组长度 |
| `athrow` | 抛出异常 |

## 六、控制流指令

### 6.1 条件分支

基于 int 值的条件跳转。

| 指令 | 说明 |
|------|------|
| `ifeq` | 等于 0 则跳转 |
| `ifne` | 不等于 0 则跳转 |
| `iflt` | 小于 0 则跳转 |
| `ifge` | 大于等于 0 则跳转 |
| `ifgt` | 大于 0 则跳转 |
| `ifle` | 小于等于 0 则跳转 |

### 6.2 比较条件分支

两个值比较的条件跳转。

| 指令 | 说明 |
|------|------|
| `if_icmpeq` | 两个 int 相等则跳转 |
| `if_icmpne` | 两个 int 不相等则跳转 |
| `if_icmplt` | int1 < int2 则跳转 |
| `if_icmpge` | int1 ≥ int2 则跳转 |
| `if_icmpgt` | int1 > int2 则跳转 |
| `if_icmple` | int1 ≤ int2 则跳转 |
| `if_acmpeq` | 两个引用相等则跳转 |
| `if_acmpne` | 两个引用不相等则跳转 |

### 6.3 null 检查分支

| 指令 | 说明 |
|------|------|
| `ifnull` | 引用为 null 则跳转 |
| `ifnonnull` | 引用不为 null 则跳转 |

### 6.4 比较指令

将比较结果压入栈。

| 指令 | 说明 |
|------|------|
| `lcmp` | 比较 long 类型（大→1, 等→0, 小→-1） |
| `fcmpl` | 比较 float（NaN 时返回 -1） |
| `fcmpg` | 比较 float（NaN 时返回 1） |
| `dcmpl` | 比较 double（NaN 时返回 -1） |
| `dcmpg` | 比较 double（NaN 时返回 1） |

### 6.5 无条件转移

| 指令 | 说明 |
|------|------|
| `goto` | 无条件跳转 |
| `goto_w` | 无条件跳转（宽索引） |

### 6.6 表跳转

多路分支跳转。

| 指令 | 说明 |
|------|------|
| `tableswitch` | 通过索引访问跳转表（连续区间） |
| `lookupswitch` | 通过键值匹配访问跳转表（离散值） |

### 6.7 子例程控制

用于 finally 块实现。

| 指令 | 说明 |
|------|------|
| `jsr` | 跳转到子例程 |
| `jsr_w` | 跳转到子例程（宽索引） |
| `ret` | 从子例程返回 |

## 七、方法调用与返回

### 7.1 方法调用

| 指令 | 说明 |
|------|------|
| `invokevirtual` | 调用对象的虚方法（动态绑定） |
| `invokespecial` | 调用特殊方法（构造器、私有方法、父类方法） |
| `invokestatic` | 调用静态方法 |
| `invokeinterface` | 调用接口方法 |

### 7.2 方法返回

| 指令 | 说明 |
|------|------|
| `ireturn` | 返回 int 类型值 |
| `lreturn` | 返回 long 类型值 |
| `freturn` | 返回 float 类型值 |
| `dreturn` | 返回 double 类型值 |
| `areturn` | 返回引用类型值 |
| `return` | void 方法返回 |

## 八、同步控制

| 指令 | 说明 |
|------|------|
| `monitorenter` | 获取对象监视器（加锁） |
| `monitorexit` | 释放对象监视器（解锁） |

## 九、指令助记符速查

### 9.1 变量与栈之间

- **装载（Load）**: `iload`, `lload`, `fload`, `dload`, `aload` + `_0`~`_3` 变体
- **存储（Store）**: `istore`, `lstore`, `fstore`, `dstore`, `astore` + `_0`~`_3` 变体

### 9.2 常量入栈

- **专用常量**: `aconst_null`, `iconst_*`, `lconst_*`, `fconst_*`, `dconst_*`
- **通用常量**: `bipush`(byte), `sipush`(short), `ldc`(常量池)

### 9.3 算术运算

- **加减乘除**: `*add`, `*sub`, `*mul`, `*div`
- **取余取反**: `*rem`, `*neg`

### 9.4 逻辑运算

- **移位**: `*shl`(左移), `*shr`(算术右移), `*ushr`(逻辑右移)
- **按位**: `*and`, `*or`, `*xor`

### 9.5 类型转换

- **拓宽**: `i2l`, `i2f`, `i2d`, `l2f`, `l2d`, `f2d`
- **缩窄**: `l2i`, `f2i`, `f2l`, `d2i`, `d2l`, `d2f`, `i2b`, `i2c`, `i2s`

### 9.6 对象与数组

- **对象**: `new`, `getfield`, `putfield`, `getstatic`, `putstatic`, `instanceof`, `checkcast`
- **数组**: `newarray`, `anewarray`, `multianewarray`, `arraylength`
- **元素访问**: `*aload`(读), `*astore`(写)

### 9.7 控制流

- **条件跳转**: `if*`, `if_icmp*`, `if_acmp*`, `ifnull`
- **比较**: `lcmp`, `fcmp*`, `dcmp*`
- **无条件**: `goto`, `goto_w`
- **多路分支**: `tableswitch`, `lookupswitch`

### 9.8 方法相关

- **调用**: `invokevirtual`, `invokespecial`, `invokestatic`, `invokeinterface`
- **返回**: `ireturn`, `lreturn`, `freturn`, `dreturn`, `areturn`, `return`

### 9.9 其他

- **栈管理**: `pop`, `pop2`, `dup`, `dup2`, `swap`
- **同步**: `monitorenter`, `monitorexit`
- **异常**: `athrow`
- **子例程**: `jsr`, `jsr_w`, `ret`
