# Flask 入门

> 目标：理解 Flask 基本概念，能够独立搭建一个可运行的 Web 后端服务

---

## 一、Flask 是什么

Flask 是一个用 Python 编写的 **轻量级 Web 框架**，核心只提供：

- **路由**（URL 映射）
- **请求 / 响应**处理
- **模板渲染**（Jinja2）

其他功能（数据库、认证、表单验证等）通过**扩展插件**按需引入，非常灵活，适合中小型后端项目和 RESTful API 开发。

## 三、第一个 Flask 应用

```python
# app.py
from flask import Flask

app = Flask(__name__)  # __name__ 告诉 Flask 当前模块位置

@app.route('/')        # 路由装饰器，映射 URL 到函数
def index():
    return 'Hello, Flask!'

if __name__ == '__main__':
    app.run(debug=True)  # debug=True 开启调试模式，代码修改后自动重启
```

运行：

```bash
python app.py
# 访问 http://127.0.0.1:5000
```

## 四、路由（Route）

### 4.1 基本路由

```python
@app.route('/hello')
def hello():
    return 'Hello World'
```

### 4.2 动态路由（URL 参数）

```python
# 字符串参数（默认）
@app.route('/user/<username>')
def get_user(username):
    return f'用户：{username}'

# 整数参数
@app.route('/post/<int:post_id>')
def get_post(post_id):
    return f'文章 ID：{post_id}'
```

|转换器|说明|
|---|---|
|`<string:name>`|默认，接受任意文本（不含 `/`）|
|`<int:id>`|整数|
|`<float:num>`|浮点数|
|`<path:subpath>`|接受含 `/` 的路径|

### 4.3 指定请求方法

```python
@app.route('/login', methods=['GET', 'POST'])
def login():
    if request.method == 'POST':
        return '处理登录'
    return '显示登录页面'
```

## 五、Request 请求对象

```python
from flask import Flask, request

@app.route('/submit', methods=['POST'])
def submit():
    # 获取表单数据
    username = request.form.get('username')

    # 获取 URL 查询参数 ?page=1
    page = request.args.get('page', 1, type=int)

    # 获取 JSON 请求体
    data = request.get_json()

    # 获取请求头
    token = request.headers.get('Authorization')

    return f'收到：{username}'
```

## 六、Response 响应对象

### 6.1 返回字符串（最简单）

```python
@app.route('/')
def index():
    return 'Hello'  # 默认状态码 200
```

### 6.2 返回 JSON（RESTful API 常用）

```python
from flask import jsonify

@app.route('/api/user')
def get_user():
    user = {'id': 1, 'name': 'Su', 'role': 'developer'}
    return jsonify(user)
    # 自动设置 Content-Type: application/json
```

### 6.3 自定义状态码和响应头

```python
@app.route('/api/create', methods=['POST'])
def create():
    return jsonify({'message': '创建成功'}), 201

# 使用 make_response 精细控制
from flask import make_response

@app.route('/custom')
def custom():
    resp = make_response(jsonify({'code': 0}), 200)
    resp.headers['X-Custom-Header'] = 'flask-app'
    return resp
```

## 七、模板渲染（Jinja2）

Flask 内置 Jinja2 模板引擎，用于渲染 HTML 页面（前后端不分离项目使用）。

### 7.1 目录结构

```
project/
├── app.py
└── templates/
    └── index.html     ← 模板文件必须放在 templates 目录
```

### 7.2 渲染模板

```python
from flask import render_template

@app.route('/page')
def page():
    return render_template('index.html', name='Su', items=[1, 2, 3])
```

### 7.3 Jinja2 模板语法

```html
<!-- templates/index.html -->
<!DOCTYPE html>
<html>
<body>
    <!-- 输出变量 -->
    <h1>Hello, {{ name }}!</h1>

    <!-- 条件判断 -->
    {% if name == 'Su' %}
        <p>欢迎管理员</p>
    {% else %}
        <p>欢迎访客</p>
    {% endif %}

    <!-- 循环 -->
    <ul>
    {% for item in items %}
        <li>{{ item }}</li>
    {% endfor %}
    </ul>
</body>
</html>
```

## 九、Flask 配置

```python
app = Flask(__name__)

# 方式一：直接设置
app.config['DEBUG'] = True
app.config['SECRET_KEY'] = 'your-secret-key'

# 方式二：从对象加载（推荐）
class Config:
    DEBUG = False
    SECRET_KEY = 'dev-secret'
    
app.config.from_object(Config)
```

常用配置项：

|配置项|说明|
|---|---|
|`DEBUG`|是否开启调试模式|
|`SECRET_KEY`|用于 Session 加密，务必设置|
|`TESTING`|测试模式开关|

## 十二、小结

|概念|关键点|
|---|---|
|路由|`@app.route()` 装饰器映射 URL|
|请求|`request.form` / `request.args` / `request.get_json()`|
|响应|`return` 字符串 / `jsonify()` / `make_response()`|
|模板|`render_template()` + Jinja2 语法|
|配置|`app.config` 字典|
