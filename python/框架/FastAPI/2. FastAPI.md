## 一、路由拆分（APIRouter）

`APIRouter` 是 FastAPI 的路由模块化方案，相当于 Flask 中的 Blueprint。

### 1.1 创建路由模块

```python
# app/routers/user.py
from fastapi import APIRouter, HTTPException
from app.schemas.user import UserCreate, UserResponse

router = APIRouter(
    prefix='/user',           # URL 前缀
    tags=['用户模块'],         # Swagger 文档分组标签
    responses={404: {'description': '资源不存在'}}
)

@router.get('/', response_model=list[UserResponse])
async def get_users():
    return []

@router.get('/{user_id}', response_model=UserResponse)
async def get_user(user_id: int):
    return {'id': user_id, 'username': 'Su'}

@router.post('/', response_model=UserResponse, status_code=201)
async def create_user(user: UserCreate):
    return user
```

### 1.2 在主入口注册路由

```python
# main.py
from fastapi import FastAPI
from app.routers import user, auth, item

app = FastAPI(title='My FastAPI Project')

app.include_router(user.router, prefix='/api')
app.include_router(auth.router, prefix='/api')
app.include_router(item.router, prefix='/api')

# 最终路径：
# GET /api/user/
# GET /api/user/{user_id}
# POST /api/auth/login
```

### 1.3 推荐项目结构

```
my-fastapi-app/
├── main.py                 # 应用入口
├── config.py               # 配置管理
├── requirements.txt
└── app/
    ├── __init__.py
    ├── database.py         # 数据库连接
    ├── dependencies.py     # 公共依赖
    ├── models/             # ORM 数据模型（SQLAlchemy）
    │   ├── __init__.py
    │   └── user.py
    ├── schemas/            # Pydantic 校验模型
    │   ├── __init__.py
    │   └── user.py
    └── routers/            # 路由模块
        ├── __init__.py
        ├── auth.py
        └── user.py
```

## 二、配置管理（pydantic-settings）

FastAPI 推荐使用 `pydantic-settings` 管理配置，支持从环境变量和 `.env` 文件自动读取。

```bash
pip install pydantic-settings
```

```python
# config.py
from pydantic_settings import BaseSettings
from functools import lru_cache

class Settings(BaseSettings):
    # 应用配置
    APP_NAME:    str = 'My FastAPI App'
    DEBUG:       bool = False
    
    # 数据库
    DB_HOST:     str = 'localhost'
    DB_PORT:     int = 3306
    DB_USER:     str = 'root'
    DB_PASSWORD: str = '123456'
    DB_NAME:     str = 'fastapi_dev'
    
    # Redis
    REDIS_HOST:  str = 'localhost'
    REDIS_PORT:  int = 6379
    
    # JWT
    JWT_SECRET:  str = 'dev-secret-key'
    JWT_EXPIRE_MINUTES: int = 120
    
    @property
    def DATABASE_URL(self) -> str:
        return f'mysql+asyncmy://{self.DB_USER}:{self.DB_PASSWORD}@{self.DB_HOST}:{self.DB_PORT}/{self.DB_NAME}'
    
    class Config:
        env_file = '.env'              # 自动读取 .env 文件
        env_file_encoding = 'utf-8'

@lru_cache()                           # 单例，避免重复加载
def get_settings() -> Settings:
    return Settings()

settings = get_settings()
```

```ini
# .env 文件（不提交 Git）
DB_PASSWORD=your_real_password
JWT_SECRET=your_real_secret
DEBUG=true
```

## 三、数据库集成（SQLAlchemy 异步版）

FastAPI 推荐使用 **异步 SQLAlchemy**，配合 `asyncmy`（MySQL）实现真正的异步数据库操作。

```bash
pip install sqlalchemy asyncmy alembic
```

### 3.1 数据库连接配置

```python
# app/database.py
from sqlalchemy.ext.asyncio import create_async_engine, AsyncSession, async_sessionmaker
from sqlalchemy.orm import DeclarativeBase
from config import settings

# 创建异步引擎
engine = create_async_engine(
    settings.DATABASE_URL,
    echo=settings.DEBUG,          # Debug 模式打印 SQL
    pool_size=10,
    max_overflow=20,
    pool_recycle=1800,
)

# 异步 Session 工厂
AsyncSessionLocal = async_sessionmaker(
    bind=engine,
    class_=AsyncSession,
    expire_on_commit=False
)

# ORM 基类
class Base(DeclarativeBase):
    pass

# 获取数据库 Session（依赖注入用）
async def get_db() -> AsyncSession:
    async with AsyncSessionLocal() as session:
        try:
            yield session
            await session.commit()
        except Exception:
            await session.rollback()
            raise
```

### 3.2 定义 ORM 模型

```python
# app/models/user.py
from sqlalchemy import String, Boolean, DateTime, Integer
from sqlalchemy.orm import Mapped, mapped_column
from datetime import datetime
from app.database import Base

class User(Base):
    __tablename__ = 'users'
    
    id:         Mapped[int]      = mapped_column(Integer, primary_key=True, index=True)
    username:   Mapped[str]      = mapped_column(String(64), unique=True, nullable=False)
    email:      Mapped[str]      = mapped_column(String(120), unique=True, nullable=False)
    password:   Mapped[str]      = mapped_column(String(255), nullable=False)
    is_active:  Mapped[bool]     = mapped_column(Boolean, default=True)
    created_at: Mapped[datetime] = mapped_column(DateTime, default=datetime.utcnow)
```

### 3.3 Pydantic Schema 与 ORM 分离

```python
# app/schemas/user.py
from pydantic import BaseModel, EmailStr, Field
from datetime import datetime
from typing import Optional

class UserCreate(BaseModel):
    username: str      = Field(..., min_length=2, max_length=20)
    email:    EmailStr
    password: str      = Field(..., min_length=6)

class UserUpdate(BaseModel):
    username: Optional[str] = None
    email:    Optional[EmailStr] = None

class UserResponse(BaseModel):
    id:         int
    username:   str
    email:      str
    is_active:  bool
    created_at: datetime
    
    model_config = {'from_attributes': True}   # 支持从 ORM 对象构建
```

### 3.4 异步 CRUD 操作

```python
# app/routers/user.py
from fastapi import APIRouter, Depends, HTTPException
from sqlalchemy.ext.asyncio import AsyncSession
from sqlalchemy import select
from app.database import get_db
from app.models.user import User
from app.schemas.user import UserCreate, UserResponse

router = APIRouter(prefix='/user', tags=['用户'])

@router.get('/{user_id}', response_model=UserResponse)
async def get_user(user_id: int, db: AsyncSession = Depends(get_db)):
    result = await db.execute(select(User).where(User.id == user_id))
    user = result.scalar_one_or_none()
    if not user:
        raise HTTPException(status_code=404, detail='用户不存在')
    return user

@router.post('/', response_model=UserResponse, status_code=201)
async def create_user(user_data: UserCreate, db: AsyncSession = Depends(get_db)):
    user = User(**user_data.model_dump())
    db.add(user)
    await db.commit()
    await db.refresh(user)
    return user
```

## 四、依赖注入（Depends）

`Depends` 是 FastAPI 最强大的特性之一，用于提取公共逻辑（数据库连接、权限校验、参数提取等）。

### 4.1 基础依赖

```python
from fastapi import Depends, Query

# 定义一个分页参数依赖
def pagination_params(
    page: int = Query(1, ge=1, description='页码'),
    size: int = Query(10, ge=1, le=100, description='每页数量')
) -> dict:
    return {'offset': (page - 1) * size, 'limit': size}

# 在路由中注入
@router.get('/')
async def get_users(
    pagination: dict = Depends(pagination_params),
    db: AsyncSession = Depends(get_db)
):
    result = await db.execute(
        select(User).offset(pagination['offset']).limit(pagination['limit'])
    )
    return result.scalars().all()
```

### 4.2 类形式的依赖（可携带状态）

```python
class QueryFilter:
    def __init__(self, keyword: str = None, is_active: bool = True):
        self.keyword   = keyword
        self.is_active = is_active

@router.get('/')
async def get_users(filters: QueryFilter = Depends()):
    # filters.keyword, filters.is_active
    pass
```

### 4.3 依赖链（依赖嵌套）

```python
# 依赖 A
async def get_db():
    ...

# 依赖 B 依赖 A
async def get_current_user(db: AsyncSession = Depends(get_db)):
    # 查数据库拿用户
    ...
    return user

# 路由依赖 B，B 自动拉起 A
@router.get('/profile')
async def profile(current_user: User = Depends(get_current_user)):
    return current_user
```

### 4.4 全局依赖（作用于所有路由）

```python
# main.py
from fastapi import FastAPI, Depends

async def verify_api_key(x_api_key: str = Header(...)):
    if x_api_key != settings.API_KEY:
        raise HTTPException(status_code=403, detail='API Key 无效')

app = FastAPI(dependencies=[Depends(verify_api_key)])   # 全局生效

# 或只作用于某个路由组
router = APIRouter(dependencies=[Depends(verify_api_key)])
```

## 五、中间件（Middleware）

### 5.1 自定义中间件

```python
import time
from fastapi import Request

@app.middleware('http')
async def logging_middleware(request: Request, call_next):
    start_time = time.time()
    
    response = await call_next(request)   # 调用下一个处理器
    
    process_time = time.time() - start_time
    response.headers['X-Process-Time'] = str(round(process_time * 1000, 2)) + 'ms'
    
    print(f'{request.method} {request.url.path} → {response.status_code} ({process_time:.3f}s)')
    return response
```

### 5.2 CORS 跨域

```python
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=['http://localhost:3000', 'https://your-frontend.com'],
    allow_credentials=True,
    allow_methods=['*'],
    allow_headers=['*'],
)
```

### 5.3 GZip 压缩

```python
from fastapi.middleware.gzip import GZipMiddleware

app.add_middleware(GZipMiddleware, minimum_size=1000)  # 超过 1KB 的响应自动压缩
```

## 六、统一响应格式封装

```python
# app/schemas/response.py
from pydantic import BaseModel
from typing import TypeVar, Generic, Optional

T = TypeVar('T')

class ApiResponse(BaseModel, Generic[T]):
    code:    int     = 0
    message: str     = 'success'
    data:    Optional[T] = None

def success(data=None, message='success') -> dict:
    return {'code': 0, 'message': message, 'data': data}

def fail(message='error', code=400) -> dict:
    return {'code': code, 'message': message, 'data': None}
```

```python
# 使用 Generic 响应模型让文档也能看到结构
@router.get('/{user_id}', response_model=ApiResponse[UserResponse])
async def get_user(user_id: int, db: AsyncSession = Depends(get_db)):
    user = ...
    return success(data=UserResponse.model_validate(user))
```

## 七、数据库迁移（Alembic）

```bash
pip install alembic

# 初始化（只执行一次）
alembic init migrations

# 修改 migrations/env.py 引入 metadata
from app.database import Base
from app.models import user   # 导入所有模型，让 Alembic 能发现表

target_metadata = Base.metadata
```

```bash
# 生成迁移脚本（每次修改模型后执行）
alembic revision --autogenerate -m "add user table"

# 应用迁移
alembic upgrade head

# 查看迁移历史
alembic history

# 回滚一个版本
alembic downgrade -1
```

## 八、启动与关闭事件

```python
# main.py（FastAPI 0.93+ 推荐使用 lifespan）
from contextlib import asynccontextmanager
from fastapi import FastAPI
from app.database import engine, Base

@asynccontextmanager
async def lifespan(app: FastAPI):
    # 启动时执行：创建数据库表
    async with engine.begin() as conn:
        await conn.run_sync(Base.metadata.create_all)
    print('✅ 应用启动')
    
    yield   # 应用运行中
    
    # 关闭时执行：释放资源
    await engine.dispose()
    print('🔴 应用关闭')

app = FastAPI(lifespan=lifespan)
```

## 九、小结

| 概念 | 关键点 |
|------|--------|
| APIRouter | 模块化路由，`prefix` + `tags` 分组 |
| pydantic-settings | 配置与环境变量管理，自动读取 `.env` |
| 异步 SQLAlchemy | `async_sessionmaker` + `AsyncSession` 真正异步 |
| Schema 分离 | ORM Model 与 Pydantic Schema 职责分离 |
| Depends | 依赖注入，复用公共逻辑，支持链式嵌套 |
| Middleware | 全局横切逻辑，CORS / 日志 / 压缩 |
| Alembic | 数据库迁移管理 |
| lifespan | 应用启动 / 关闭生命周期钩子 |
