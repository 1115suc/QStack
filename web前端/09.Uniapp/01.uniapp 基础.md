# UniApp 基础篇

> 适用版本：uni-app 3.x + Vue 3 + Vite + TypeScript

---

## 1. UniApp 简介

UniApp 是 DCloud 出品的跨平台前端框架，**一套代码**可编译到以下平台：

| 平台 | 说明 |
|------|------|
| H5 | 浏览器网页 |
| 微信小程序 | 微信生态 |
| 支付宝小程序 | 支付宝生态 |
| App（Android/iOS） | 基于 WebView 或原生渲染 |
| 抖音小程序 | 字节跳动生态 |
| 快应用 | 华为/小米等厂商 |

### 与 Vue3 关系

- UniApp 3.x 支持 Vue3 的 **Composition API** 和 `<script setup>` 语法
- 不支持 Vue3 的部分特性（如 `Teleport`、`Suspense`）
- 推荐使用 **Vite** 构建（比 webpack 快）

### 与普通 Vue 项目的主要区别

- 页面路由由 `pages.json` 管理，不使用 vue-router
- 没有 `document`、`window` 等 DOM/BOM API
- 使用 `uni.*` 系列 API 替代浏览器 API
- 样式单位推荐 `rpx`，不能使用部分 CSS 特性

## 2. 项目创建与目录结构

### 2.1 创建项目

```bash
# 方式一：HBuilderX（官方 IDE，推荐新手）
# 直接在 HBuilderX 中 新建项目 → uni-app → Vue3

# 方式二：CLI 创建（推荐配合 VSCode）
npx degit dcloudio/uni-preset-vue#vite-ts my-project
cd my-project
npm install
npm run dev:mp-weixin   # 运行到微信小程序
npm run dev:h5          # 运行到 H5
```

### 2.2 标准目录结构

```
my-project/
├── src/
│   ├── pages/                  # 页面文件（每个页面一个文件夹）
│   │   ├── index/
│   │   │   └── index.vue
│   │   └── user/
│   │       └── index.vue
│   ├── components/             # 公共组件
│   ├── composables/            # Vue3 组合式函数
│   ├── stores/                 # Pinia 状态管理
│   ├── utils/                  # 工具函数
│   ├── api/                    # 接口请求
│   ├── static/                 # 静态资源（图片等，不会被编译）
│   ├── uni_modules/            # uni-app 插件市场组件
│   ├── App.vue                 # 应用入口（配置全局样式、生命周期）
│   ├── main.ts                 # 入口文件
│   ├── pages.json              # 页面路由配置（核心！）
│   ├── manifest.json           # 应用配置（AppID、权限等）
│   └── uni.scss                # 全局 SCSS 变量
├── vite.config.ts
└── tsconfig.json
```

### 2.3 main.ts 入口

```typescript
import { createSSRApp } from 'vue'
import App from './App.vue'
import { createPinia } from 'pinia'

export function createApp() {
  const app = createSSRApp(App)
  const pinia = createPinia()
  app.use(pinia)
  return { app, pinia }
}
```

> ⚠️ 注意：UniApp 使用 `createSSRApp` 而非 `createApp`，不要替换。

## 3. 页面与路由

### 3.1 pages.json 配置

这是 UniApp 路由的核心文件，**所有页面必须在此注册**。

```json
{
  "pages": [
    {
      "path": "pages/index/index",
      "style": {
        "navigationBarTitleText": "首页",
        "navigationBarBackgroundColor": "#ffffff",
        "navigationBarTextStyle": "black"
      }
    },
    {
      "path": "pages/user/index",
      "style": {
        "navigationBarTitleText": "个人中心"
      }
    }
  ],
  "globalStyle": {
    "navigationBarTextStyle": "black",
    "navigationBarTitleText": "MyApp",
    "navigationBarBackgroundColor": "#F8F8F8",
    "backgroundColor": "#F8F8F8"
  },
  "tabBar": {
    "color": "#7A7E83",
    "selectedColor": "#3cc51f",
    "borderStyle": "black",
    "backgroundColor": "#ffffff",
    "list": [
      {
        "pagePath": "pages/index/index",
        "iconPath": "static/tab-home.png",
        "selectedIconPath": "static/tab-home-active.png",
        "text": "首页"
      },
      {
        "pagePath": "pages/user/index",
        "iconPath": "static/tab-user.png",
        "selectedIconPath": "static/tab-user-active.png",
        "text": "我的"
      }
    ]
  }
}
```

### 3.2 页面跳转 API

```typescript
// 跳转到新页面（可返回）
uni.navigateTo({ url: '/pages/user/index' })

// 跳转并关闭当前页（不可返回）
uni.redirectTo({ url: '/pages/login/index' })

// 关闭所有页面，跳转到指定页
uni.reLaunch({ url: '/pages/index/index' })

// 返回上一页
uni.navigateBack({ delta: 1 })

// 跳转到 tabBar 页面（必须用此方法）
uni.switchTab({ url: '/pages/index/index' })
```

### 3.3 路由传参

```typescript
// 发送方：query 字符串传参
uni.navigateTo({
  url: `/pages/detail/index?id=${item.id}&name=${encodeURIComponent(item.name)}`
})

// 接收方：onLoad 中获取（注意：这是 UniApp 生命周期，不是 Vue 生命周期）
// options 就是路由参数对象
onLoad((options) => {
  const id = options?.id
  const name = decodeURIComponent(options?.name ?? '')
})
```

> 💡 传递对象参数时，需要先 `JSON.stringify` 再 `encodeURIComponent`，接收时反向操作。

### 3.4 使用 uni-mini-router（推荐）

官方路由较原始，项目稍大建议引入 [uni-mini-router](https://github.com/Moonofweisheng/uni-mini-router) 实现类 vue-router 的体验。

## 4. 基础组件

UniApp 提供了一套内置基础组件，**不能直接使用 HTML 标签**（H5 可以但不推荐）。

### 4.1 常用基础组件对照

| HTML 标签 | UniApp 组件 | 说明 |
|-----------|------------|------|
| `<div>` | `<view>` | 块级容器 |
| `<span>` | `<text>` | 行内文本 |
| `<img>` | `<image>` | 图片，默认宽 320px 高 240px |
| `<input>` | `<input>` | 同名但属性有差异 |
| `<textarea>` | `<textarea>` | 多行输入 |
| `<a>` | `<navigator>` | 页面跳转链接 |
| `<video>` | `<video>` | 视频播放 |
| `<scroll-view>` | `<scroll-view>` | 可滚动视图 |

### 4.2 view 与 text

```html
<template>
  <!-- view：块级容器，相当于 div -->
  <view class="container">
    <!-- text：文本组件，支持 selectable（可选中复制） -->
    <text selectable>可以长按复制这段文字</text>

    <!-- 注意：text 内只能嵌套 text，不能嵌套 view -->
    <text>用户名：<text style="color: red;">张三</text></text>
  </view>
</template>
```

### 4.3 image 组件

```html
<template>
  <!-- mode 控制图片裁剪/缩放 -->
  <image
    src="/static/logo.png"
    mode="aspectFill"
    style="width: 100rpx; height: 100rpx;"
    @error="onError"
    @load="onLoad"
  />
</template>
```

**常用 mode 值：**

| mode | 说明 |
|------|------|
| `scaleToFill` | 默认，拉伸填满（变形） |
| `aspectFit` | 保持比例，完整显示（可能有空白） |
| `aspectFill` | 保持比例，填满裁剪（推荐） |
| `widthFix` | 宽度固定，高度自适应 |

### 4.4 scroll-view 滚动容器

```html
<template>
  <!-- 纵向滚动，必须设置固定高度 -->
  <scroll-view
    scroll-y
    style="height: 400rpx;"
    @scrolltolower="loadMore"
    :refresher-enabled="true"
    @refresherrefresh="onRefresh"
  >
    <view v-for="item in list" :key="item.id">
      {{ item.name }}
    </view>
  </scroll-view>
</template>
```

### 4.5 input 组件

```html
<template>
  <input
    v-model="keyword"
    type="text"
    placeholder="请输入关键词"
    placeholder-style="color: #999;"
    :maxlength="50"
    @confirm="onSearch"
    @focus="onFocus"
  />
</template>
```

**常用 type 值：** `text`、`number`、`digit`（带小数的数字）、`tel`、`safe-password`

## 5. 样式与尺寸单位

### 5.1 rpx 单位

rpx（responsive pixel）是 UniApp 推荐的响应式单位，以 **750rpx = 屏幕宽度** 为基准自动换算。

```css
/* 设计稿基于 750px 宽度时，1px = 1rpx */
.box {
  width: 200rpx;    /* 在不同设备自动换算 */
  height: 100rpx;
  font-size: 28rpx;
  border-radius: 16rpx;
}
```

> 💡 设计稿如果是 375px 宽，则所有标注尺寸 × 2 即可得到 rpx 值。

### 5.2 样式注意事项

```css
/* ✅ 支持 */
.container {
  display: flex;
  flex-direction: column;
  background-color: #f5f5f5;
  border-radius: 16rpx;
}

/* ❌ 不支持（小程序环境）*/
.item:last-child { }         /* 部分伪类不支持 */
.parent > .child { }         /* 子选择器不支持（H5 可以）*/
[data-type="btn"] { }        /* 属性选择器不支持 */
* { }                         /* 通配符选择器不支持 */
```

### 5.3 全局样式

```scss
// uni.scss —— 全局 SCSS 变量，所有页面自动可用，无需 import
$primary-color: #3B82F6;
$success-color: #22C55E;
$danger-color: #EF4444;
$text-color: #1f2937;
$text-secondary: #6b7280;
$border-radius: 16rpx;
$page-padding: 24rpx;
```

```css
/* App.vue —— 全局基础样式 */
<style>
page {
  background-color: #f5f5f5;
  font-size: 28rpx;
  color: #1f2937;
  box-sizing: border-box;
}
</style>
```

## 6. 生命周期

UniApp 的生命周期分为三类，注意区分。

### 6.1 应用生命周期（App.vue）

```typescript
// App.vue
<script setup lang="ts">
// 应用生命周期只能在 App.vue 中使用
onLaunch((options) => {
  // 应用初始化完成，全局只触发一次
  console.log('App Launch', options)
})

onShow((options) => {
  // 应用从后台切换到前台
})

onHide(() => {
  // 应用切换到后台
})
</script>
```

### 6.2 页面生命周期

```typescript
<script setup lang="ts">
import { ref } from 'vue'

const list = ref([])

// ✅ UniApp 页面生命周期（直接从 'uni-app' 导入，或全局可用）
onLoad((options) => {
  // 页面加载，可获取路由参数
  console.log('页面参数:', options)
  fetchData()
})

onShow(() => {
  // 每次页面显示都会触发（包括从下一页返回）
  // 适合刷新列表
})

onReady(() => {
  // 页面初次渲染完成（DOM 可操作）
})

onHide(() => {
  // 页面隐藏（跳转到下一页时触发）
})

onUnload(() => {
  // 页面卸载（redirectTo / navigateBack 时触发）
  // 清理定时器、取消请求等
})

// 下拉刷新（需要在 pages.json 对应页面开启 enablePullDownRefresh: true）
onPullDownRefresh(async () => {
  await fetchData()
  uni.stopPullDownRefresh()
})

// 上拉加载更多
onReachBottom(() => {
  loadMore()
})

// 页面滚动
onPageScroll((e) => {
  console.log('scrollTop:', e.scrollTop)
})
</script>
```

### 6.3 组件生命周期（与 Vue3 相同）

```typescript
// 普通组件内，使用 Vue3 的生命周期钩子
import { onMounted, onUnmounted, onActivated } from 'vue'

onMounted(() => {
  // 组件挂载完成
})

onUnmounted(() => {
  // 组件销毁（清理副作用）
})
```

### 6.4 生命周期执行顺序总结

```
应用启动：
App onLaunch → App onShow

首次进入页面：
Page onLoad → Page onShow → Vue onMounted → Page onReady

跳转到下一页：
当前页 onHide → 下一页 onLoad → 下一页 onShow → 下一页 onReady

从下一页返回：
下一页 onUnload → 当前页 onShow
```

## 7. 数据绑定与事件

### 7.1 模板语法（与 Vue3 基本一致）

```html
<template>
  <view>
    <!-- 文本插值 -->
    <text>{{ title }}</text>

    <!-- 属性绑定 -->
    <image :src="imgUrl" />

    <!-- 条件渲染 -->
    <view v-if="isLogin">已登录内容</view>
    <view v-else>请登录</view>

    <!-- v-show（通过 display 控制，不销毁 DOM）-->
    <view v-show="visible">弹窗内容</view>

    <!-- 列表渲染（必须加 key！）-->
    <view v-for="item in list" :key="item.id">
      {{ item.name }}
    </view>
  </view>
</template>
```

### 7.2 事件绑定

```html
<template>
  <view>
    <!-- 点击事件 -->
    <view @tap="handleTap">点击我</view>

    <!-- 带参数 -->
    <view @tap="handleTap($event, item.id)">点击</view>

    <!-- 阻止冒泡（小程序中用 @tap.stop）-->
    <view @tap.stop="handleTap">阻止冒泡</view>

    <!-- 长按事件 -->
    <view @longpress="handleLongPress">长按</view>

    <!-- 表单输入 -->
    <input @input="onInput" @blur="onBlur" @confirm="onConfirm" />
  </view>
</template>

<script setup lang="ts">
// 小程序的点击事件对象类型
function handleTap(event: UniHelper.BaseEvent) {
  console.log(event.detail)
}
</script>
```

> ⚠️ 小程序中点击事件是 `@tap`，H5 中 `@click` 也可以，但统一用 `@tap` 兼容性更好。

### 7.3 双向绑定

```html
<template>
  <!-- input 的 v-model -->
  <input v-model="username" placeholder="用户名" />

  <!-- 自定义组件 v-model（Vue3 写法）-->
  <MyInput v-model="value" />
</template>

<script setup lang="ts">
import { ref } from 'vue'
const username = ref('')
</script>
```

### 7.4 计算属性与侦听器

```typescript
<script setup lang="ts">
import { ref, computed, watch } from 'vue'

const price = ref(100)
const count = ref(2)

// 计算属性
const total = computed(() => price.value * count.value)

// 侦听单个值
watch(count, (newVal, oldVal) => {
  console.log(`数量从 ${oldVal} 变为 ${newVal}`)
})

// 深度侦听对象
const form = ref({ name: '', age: 0 })
watch(form, (newVal) => {
  console.log('表单变化', newVal)
}, { deep: true })
</script>
```

## 总结

| 知识点 | 关键记忆 |
|--------|---------|
| 路由 | `pages.json` 管理，`uni.navigateTo` 跳转 |
| 组件 | `view` / `text` / `image`，不用 HTML 标签 |
| 单位 | `rpx`，750rpx = 屏幕宽 |
| 生命周期 | 页面用 `onLoad/onShow`，组件用 `onMounted` |
| 事件 | 点击用 `@tap`，阻止冒泡用 `.stop` |
