# CSS基础学习笔记

## 一、CSS基础认知

### 1.1 CSS简介
- **全称**: CSS (Cascading Style Sheets) - 层叠样式表
- **作用**: 给HTML标签设置样式,美化网页
- **Web标准三层**:
    - 结构层: HTML (决定身体)
    - 表现层: CSS (决定样式美观)
    - 行为层: JavaScript (决定交互动态效果)

### 1.2 CSS语法规则

```css
选择器 {
    属性名: 属性值;
    属性名: 属性值;
}
```

**注意事项**:
1. CSS标点符号必须是**英文状态**
2. 每个样式声明以**分号**结尾
3. 属性名和属性值之间用**冒号**分隔

### 1.3 CSS引入方式

#### 1. 内嵌式 (推荐学习使用)
```html
<head>
    <style>
        p {
            color: red;
        }
    </style>
</head>
```
- **特点**: CSS写在`<style>`标签中
- **位置**: 通常写在`<head>`标签内
- **作用范围**: 当前HTML页面
- **使用场景**: 小型页面、练习

#### 2. 外联式 (推荐开发使用)
```html
<head>
    <link rel="stylesheet" href="./css/style.css">
</head>
```
- **特点**: CSS写在独立的`.css`文件中
- **引入方式**: 通过`<link>`标签引入
- **作用范围**: 多个HTML页面
- **使用场景**: 项目开发(便于维护和复用)

#### 3. 行内式 (不推荐)
```html
<div style="color: red; font-size: 20px;">内容</div>
```
- **特点**: CSS写在标签的`style`属性中
- **作用范围**: 当前标签
- **使用场景**: 配合JavaScript动态修改样式

## 二、基础选择器

### 2.1 选择器的作用
- **目的**: 选中页面中的标签(找她),设置样式(改她)

### 2.2 标签选择器

```css
p {
    color: red;
}
```

**语法**: `标签名 { 属性: 值; }`

**特点**:
- 选中页面中**所有**该类型标签
- 无论嵌套多深都能找到
- 无法差异化选择

### 2.3 类选择器 ⭐(最常用)

```css
.box {
    color: blue;
}
```

```html
<div class="box">内容</div>
<p class="box">文字</p>
```

**语法**: `.类名 { 属性: 值; }`

**特点**:
1. 所有标签都有`class`属性
2. 类名可由**字母、数字、下划线、中划线**组成
3. 不能以**数字或中划线**开头
4. 一个标签可以有**多个类名**(空格隔开)
5. 类名可以**重复使用**

**多类名使用**:
```html
<div class="box red large">内容</div>
```

```css
.box { width: 200px; }
.red { color: red; }
.large { font-size: 20px; }
```

### 2.4 id选择器

```css
#header {
    background-color: skyblue;
}
```

```html
<div id="header">头部</div>
```

**语法**: `#id值 { 属性: 值; }`

**特点**:
1. 所有标签都有`id`属性
2. id值在页面中**必须唯一**(类似身份证号)
3. 一个标签只能有**一个**id值
4. 一个id选择器只能选中**一个**标签

**class vs id**:
| 对比项 | class类名 | id属性值 |
|--------|----------|----------|
| 符号 | `.` | `#` |
| 复用性 | 可重复使用 | 页面唯一 |
| 数量 | 一个标签可多个 | 一个标签一个 |
| 类比 | 姓名 | 身份证号 |
| 使用频率 | ⭐⭐⭐⭐⭐ | ⭐(配合JS使用) |

### 2.5 通配符选择器

```css
* {
    margin: 0;
    padding: 0;
}
```

**语法**: `* { 属性: 值; }`

**特点**:
- 选中页面**所有**标签
- 性能较差,使用较少
- 常用于清除默认边距

## 三、字体样式属性

### 3.1 字体大小 font-size

```css
p {
    font-size: 16px;
}
```

**属性值**: 数字 + px

**注意**:
- 谷歌浏览器默认字体大小: **16px**
- 必须带单位,否则无效

### 3.2 字体粗细 font-weight

```css
h1 {
    font-weight: normal;  /* 取消加粗 */
}
p {
    font-weight: bold;    /* 加粗 */
}
```

**常用取值**:
| 值 | 效果 | 说明 |
|-----|------|------|
| `normal` | 正常 | 等同于400 |
| `bold` | 加粗 | 等同于700 |
| `400` | 正常 | 数值形式 |
| `700` | 加粗 | 数值形式 |

**可选值**: 100~900的整百数

**注意**: 不是所有字体都提供9种粗细

### 3.3 字体样式 font-style

```css
em {
    font-style: normal;  /* 取消倾斜 */
}
p {
    font-style: italic;  /* 倾斜 */
}
```

**取值**:
- `normal`: 正常(默认)
- `italic`: 倾斜

### 3.4 字体系列 font-family

```css
body {
    font-family: "Microsoft YaHei", "微软雅黑", sans-serif;
}
```

**语法**: `font-family: 字体1, 字体2, 字体系列;`

**三大字体系列**:
1. **无衬线字体** (sans-serif) - 推荐网页使用
    - 特点: 笔画粗细均匀,无装饰
    - 示例: 黑体、Arial、微软雅黑

2. **衬线字体** (serif) - 适合印刷
    - 特点: 笔画粗细不均,有笔锋装饰
    - 示例: 宋体、Times New Roman

3. **等宽字体** (monospace) - 适合代码
    - 特点: 每个字符宽度相等
    - 示例: Consolas、Fira Code

**渲染规则**:
1. 从左到右查找字体
2. 找不到则使用下一个
3. 都找不到则使用字体系列的默认字体

**注意事项**:
1. 多个单词的字体名用**引号包裹**
2. 最后的字体系列**不加引号**
3. 优先使用系统自带字体

### 3.5 字体复合属性 font

```css
p {
    /* font: style weight size/line-height family; */
    font: italic 700 16px/1.5 "微软雅黑", sans-serif;
}
```

**语法**: `font: [style] [weight] size[/line-height] family;`

**注意**:
1. `size`和`family`**不能省略**
2. 省略的属性相当于设置默认值
3. 必须按照顺序书写

**层叠问题**:
- 相同属性后面覆盖前面
- 单独样式要写在font连写**后面**

## 四、文本样式属性

### 4.1 文本缩进 text-indent

```css
p {
    text-indent: 2em;  /* 首行缩进2个字符 */
}
```

**取值**:
- `数字 + px`: 固定像素
- `数字 + em`: **推荐** (1em = 当前字体大小)

### 4.2 文本对齐 text-align

```css
h1 {
    text-align: center;
}
```

**取值**:
| 值 | 效果 |
|----|------|
| `left` | 左对齐(默认) |
| `center` | 居中对齐 |
| `right` | 右对齐 |

**能让什么水平居中**:
1. 文本
2. `span`、`a`标签
3. `input`、`img`标签

**注意**: 给**父元素**设置,子元素生效

### 4.3 文本修饰 text-decoration

```css
a {
    text-decoration: none;  /* 去除下划线 */
}
h1 {
    text-decoration: underline;  /* 添加下划线 */
}
```

**取值**:
| 值 | 效果 |
|----|------|
| `none` | 无装饰 |
| `underline` | 下划线 |
| `line-through` | 删除线 |
| `overline` | 上划线 |

**常用场景**: 清除`<a>`标签默认下划线

### 4.4 颜色属性 color

```css
p {
    color: red;                    /* 关键词 */
    color: rgb(255, 0, 0);         /* RGB */
    color: rgba(255, 0, 0, 0.5);   /* RGBA */
    color: #ff0000;                /* 十六进制 */
    color: #f00;                   /* 简写 */
}
```

**四种表示法**:

#### 1. 关键词
```css
color: red;      /* 红色 */
color: blue;     /* 蓝色 */
color: green;    /* 绿色 */
color: skyblue;  /* 天蓝色 */
```

#### 2. RGB表示法
```css
color: rgb(255, 0, 0);    /* 红色 */
color: rgb(0, 255, 0);    /* 绿色 */
color: rgb(0, 0, 255);    /* 蓝色 */
color: rgb(0, 0, 0);      /* 黑色 */
color: rgb(255, 255, 255); /* 白色 */
```
- 取值范围: 0~255

#### 3. RGBA表示法
```css
color: rgba(0, 0, 0, 0.5);  /* 半透明黑色 */
color: rgba(0, 0, 0, .5);   /* 省略写法 */
```
- a表示透明度: 0(完全透明) ~ 1(完全不透明)

#### 4. 十六进制表示法 (最常用)
```css
color: #ff0000;  /* 红色 */
color: #f00;     /* 简写(每组相同可省略) */
color: #000;     /* 黑色 */
color: #fff;     /* 白色 */
```
- 取值: 0~9, a~f
- 三组两位数,每组相同可简写

## 五、行高 line-height

```css
p {
    line-height: 30px;   /* 固定值 */
    line-height: 1.5;    /* 倍数 */
    line-height: 2;      /* 当前字体的2倍 */
}
```

**作用**: 控制一行的上下间距

**取值**:
- `数字 + px`: 固定行高
- `倍数`: 当前font-size的倍数

**应用场景**:

### 5.1 单行文本垂直居中
```css
div {
    height: 100px;
    line-height: 100px;  /* 行高 = 盒子高度 */
}
```

### 5.2 取消上下间距
```css
p {
    line-height: 1;  /* 精准布局 */
}
```

### 5.3 font连写中包含行高
```css
p {
    font: 16px/1.5 "微软雅黑";
    /* 等同于 */
    font-size: 16px;
    line-height: 1.5;
    font-family: "微软雅黑";
}
```

## 六、元素水平居中方法

### 6.1 文本/行内元素居中
```css
/* 给父元素设置 */
.parent {
    text-align: center;
}
```

**适用于**:
- 文本
- `span`、`a`
- `img`、`input`

### 6.2 块级元素居中
```css
/* 给自己设置 */
.box {
    width: 800px;
    margin: 0 auto;
}
```

**注意**:
- 必须有**固定宽度**
- 给**元素本身**设置

## 八、CSS样式层叠

### 8.1 层叠规则
```css
p {
    color: red;
    color: blue;  /* 生效 */
}
```

**规则**:
- 相同属性,**后面覆盖前面**
- 不同属性,**都会生效**

### 8.2 样式优先级(初级)
```css
/* 同一个选择器内 */
p {
    color: red;
    font-size: 20px;
    color: blue;  /* color生效这个 */
}
```

## 九、开发规范与技巧

### 9.1 命名规范
```css
/* 推荐 */
.news-title { }
.user-info { }

/* 不推荐 */
.newsTitle { }  /* 小驼峰 */
.NewsTitle { }  /* 大驼峰 */
```

### 9.2 代码复用
```css
/* 公共样式抽取 */
.text-center { text-align: center; }
.color-red { color: red; }

/* 多个类名组合使用 */
<p class="text-center color-red">内容</p>
```

### 9.3 选择器使用建议
1. **优先使用类选择器** (最灵活)
2. id选择器配合JavaScript使用
3. 标签选择器用于通用样式
4. 通配符选择器谨慎使用(性能)

## 十、常见问题总结

### 10.1 字体相关问题
❌ **错误**:
```css
p {
    font-size: 16;  /* 缺少单位 */
}
```

✅ **正确**:
```css
p {
    font-size: 16px;
}
```

### 10.2 选择器问题
❌ **错误**:
```css
<div class="box"></div>
#box { }  /* 类名用了id选择器 */
```

✅ **正确**:
```css
<div class="box"></div>
.box { }
```

### 10.3 居中问题
❌ **错误**:
```css
/* img自己设置 */
img {
    text-align: center;  /* 无效 */
}
```

✅ **正确**:
```css
/* 给img的父元素设置 */
.parent {
    text-align: center;
}
```
