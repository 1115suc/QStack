## 一、选择器进阶

### 1.1 复合选择器

#### 1.1.1 后代选择器 (空格)

**定义**: 根据HTML标签的嵌套关系,选择父元素**所有后代**中满足条件的元素

**语法**:
```css
选择器1 选择器2 {
    CSS属性: 属性值;
}
```

**特点**:
- ✅ 选中所有后代(儿子、孙子、重孙子...)
- ✅ 选择器之间用**空格**隔开
- ✅ 无论嵌套多深都能找到
- ✅ 可以是任意基础选择器的组合

**代码示例**:
```html
<div class="container">
    <p>我会变红</p>
    <div>
        <p>我也会变红</p>
        <ul>
            <li>
                <p>我还会变红</p>
            </li>
        </ul>
    </div>
</div>
```

```css
/* 选中 .container 内所有 p 标签 */
.container p {
    color: red;
}

/* 选中 div 内所有 span */
div span {
    font-size: 14px;
}

/* 多层嵌套 */
.nav ul li a {
    text-decoration: none;
}
```

**应用场景**:
```css
/* 导航栏链接 */
.nav a {
    color: #333;
}

/* 列表项样式 */
.news-list li {
    margin-bottom: 10px;
}

/* 文章段落 */
.article p {
    text-indent: 2em;
    line-height: 1.8;
}
```

#### 1.1.2 子代选择器 (>)

**定义**: 根据HTML标签的嵌套关系,选择父元素**直接子代**中满足条件的元素

**语法**:
```css
选择器1 > 选择器2 {
    CSS属性: 属性值;
}
```

**特点**:
- ✅ 只选中**直接子元素**(儿子)
- ✅ 选择器之间用 **>** 隔开
- ❌ 孙子、重孙子等不会被选中
- ✅ 更精准的选择

**代码示例**:
```html
<div class="box">
    <p>我是儿子,会变蓝</p>
    <div>
        <p>我是孙子,不会变蓝</p>
    </div>
</div>
```

```css
/* 只选中 .box 的直接子元素 p */
.box > p {
    color: blue;
}

/* 选中 ul 的直接子元素 li */
.menu > li {
    float: left;
}
```

**后代选择器 vs 子代选择器对比**:

| 对比项 | 后代选择器(空格) | 子代选择器(>) |
|--------|------------------|---------------|
| 符号 | 空格 | `>` |
| 选择范围 | 所有后代 | 仅直接子代 |
| 选择深度 | 无限深 | 仅一层 |
| 精准度 | 较低 | 较高 |
| 使用频率 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

**实战对比**:
```html
<ul class="list">
    <li>一级列表项
        <ul>
            <li>二级列表项</li>
        </ul>
    </li>
</ul>
```

```css
/* 后代选择器:一级和二级都选中 */
.list li {
    color: red;
}

/* 子代选择器:只选中一级 */
.list > li {
    color: blue;
}
```

### 1.2 并集选择器 (,)

**定义**: 同时选择多组标签,给它们设置相同的样式

**语法**:
```css
选择器1,
选择器2,
选择器3 {
    CSS属性: 属性值;
}
```

**特点**:
- ✅ 同时选中**多组**元素
- ✅ 选择器之间用**逗号**隔开
- ✅ 每组选择器通常**单独一行**(提高可读性)
- ✅ 可以是基础选择器或复合选择器
- ✅ 减少代码冗余

**代码示例**:
```css
/* 同时选中多个标签 */
div,
p,
span {
    color: red;
}

/* 同时选中多个类 */
.header,
.footer,
.sidebar {
    background-color: #f5f5f5;
}

/* 混合使用 */
h1,
h2,
h3,
.title {
    font-weight: normal;
}

/* 复合选择器组合 */
.nav a,
.menu a,
.footer a {
    text-decoration: none;
    color: #333;
}
```

**实际应用**:
```css
/* 清除默认样式 */
body,
h1, h2, h3, h4, h5, h6,
p,
ul, ol, li {
    margin: 0;
    padding: 0;
}

/* 统一标题样式 */
h1,
h2,
h3 {
    font-weight: 500;
    color: #333;
    line-height: 1.2;
}

/* 统一输入框样式 */
input,
textarea,
select {
    border: 1px solid #ddd;
    padding: 5px 10px;
    font-size: 14px;
}
```

**注意事项**:
1. 每个选择器独立一行,提高可读性
2. 最后一个选择器后不要加逗号
3. 建议相关性强的选择器分组

### 1.3 交集选择器 (紧挨着)

**定义**: 选中页面中**同时满足**多个选择器条件的标签(既又原则)

**语法**:
```css
选择器1选择器2 {
    CSS属性: 属性值;
}
```

**特点**:
- ✅ 选择器之间**紧挨着**,无任何符号
- ✅ 必须**同时满足**所有条件
- ⚠️ 如果有标签选择器,必须**写在最前面**
- ✅ 提高选择器精准度

**代码示例**:
```html
<p class="important">我会变红(既是p又有important类)</p>
<p>我不会变红(只是p)</p>
<div class="important">我不会变红(只有important类)</div>
```

```css
/* 选中既是p标签又有important类的元素 */
p.important {
    color: red;
}

/* 选中同时有box和red类的元素 */
.box.red {
    background-color: pink;
}

/* 错误写法 */
.important p {  /* 这是后代选择器,不是交集选择器 */
    color: blue;
}
```

**实际应用**:
```html
<button class="btn primary">主要按钮</button>
<button class="btn">普通按钮</button>
```

```css
/* 基础按钮样式 */
.btn {
    padding: 10px 20px;
    border: none;
    cursor: pointer;
}

/* 主要按钮额外样式 */
.btn.primary {
    background-color: #007bff;
    color: white;
}
```

**标签选择器必须在前**:
```css
/* ✅ 正确 */
p.box {
    color: red;
}

/* ❌ 错误 */
.box p {  /* 这变成了后代选择器 */
    color: red;
}
```

### 1.4 伪类选择器

#### 1.4.1 :hover 鼠标悬停伪类

**定义**: 选中鼠标悬停在元素上的**状态**,设置悬停样式

**语法**:
```css
选择器:hover {
    CSS属性: 属性值;
}
```

**特点**:
- ✅ 选中元素的**特定状态**
- ✅ 冒号与选择器**紧挨着**
- ✅ 任何元素都可以使用
- ✅ 常用于交互效果

**基础示例**:
```css
/* 链接悬停 */
a:hover {
    color: red;
}

/* 按钮悬停 */
button:hover {
    background-color: #0056b3;
}

/* 盒子悬停 */
.box:hover {
    box-shadow: 0 0 10px rgba(0,0,0,0.3);
}
```

**导航栏悬停效果**:
```html
<nav class="nav">
    <a href="#">首页</a>
    <a href="#">产品</a>
    <a href="#">关于</a>
</nav>
```

```css
.nav a {
    display: inline-block;
    padding: 10px 20px;
    color: #333;
    text-decoration: none;
    transition: all 0.3s;
}

.nav a:hover {
    color: #ff6700;
    background-color: #f5f5f5;
    border-bottom: 2px solid #ff6700;
}
```

**图片悬停效果**:
```css
/* 图片放大 */
img:hover {
    transform: scale(1.1);
    transition: transform 0.3s;
}

/* 图片变暗 */
.photo:hover {
    opacity: 0.8;
}

/* 图片旋转 */
.icon:hover {
    transform: rotate(360deg);
    transition: transform 0.5s;
}
```

**按钮悬停效果**:
```css
.btn {
    background-color: #007bff;
    color: white;
    padding: 10px 30px;
    border: none;
    cursor: pointer;
    transition: all 0.3s;
}

.btn:hover {
    background-color: #0056b3;
    transform: translateY(-2px);
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
}
```

**卡片悬停效果**:
```css
.card {
    border: 1px solid #ddd;
    padding: 20px;
    transition: all 0.3s;
}

.card:hover {
    box-shadow: 0 5px 15px rgba(0,0,0,0.1);
    transform: translateY(-5px);
}
```

**复杂悬停效果**:
```css
/* 子元素联动 */
.item:hover .title {
    color: #ff6700;
}

.item:hover .image {
    transform: scale(1.05);
}

/* 背景渐变 */
.box {
    background: linear-gradient(to right, #667eea 0%, #764ba2 100%);
    transition: opacity 0.3s;
}

.box:hover {
    opacity: 0.9;
}
```

### 1.5 选择器总结对比

| 选择器 | 符号 | 作用 | 语法示例 | 使用频率 |
|--------|------|------|----------|----------|
| 后代选择器 | 空格 | 选中所有后代 | `div p` | ⭐⭐⭐⭐⭐ |
| 子代选择器 | `>` | 只选中儿子 | `div > p` | ⭐⭐⭐ |
| 并集选择器 | `,` | 同时选中多个 | `div, p` | ⭐⭐⭐⭐ |
| 交集选择器 | 紧挨 | 同时满足条件 | `p.box` | ⭐⭐ |
| 伪类选择器 | `:` | 选中特定状态 | `a:hover` | ⭐⭐⭐⭐⭐ |

## 二、Emmet语法

### 2.1 什么是Emmet

**定义**: 通过简写语法,快速生成HTML和CSS代码的工具

**优势**:
- ⚡ 提高编码速度
- ✅ 减少重复劳动
- 🎯 代码更简洁
- 📝 支持复杂结构

### 2.2 HTML快速生成

#### 2.2.1 基础标签生成

```css
div              →  <div></div>
p                →  <p></p>
span             →  <span></span>
```

#### 2.2.2 类名和ID

```css
div.box          →  <div class="box"></div>
div#header       →  <div id="header"></div>
p.text.red       →  <p class="text red"></p>
div.box#main     →  <div class="box" id="main"></div>
```

#### 2.2.3 乘法(生成多个)

```css
ul>li*5          →  <ul>
                       <li></li>
                       <li></li>
                       <li></li>
                       <li></li>
                       <li></li>
                   </ul>

div.box*3        →  <div class="box"></div>
                    <div class="box"></div>
                    <div class="box"></div>
```

#### 2.2.4 子元素(>)

```css
div>p            →  <div>
                       <p></p>
                   </div>

ul>li>a          →  <ul>
                       <li><a href=""></a></li>
                   </ul>
```

#### 2.2.5 兄弟元素(+)

```css
div+p            →  <div></div>
                    <p></p>

h1+p+p           →  <h1></h1>
                    <p></p>
                    <p></p>
```

#### 2.2.6 上级元素(^)

```css
div>p^div        →  <div>
                       <p></p>
                   </div>
                   <div></div>
```

#### 2.2.7 分组(())

```css
(div>p)*3        →  <div><p></p></div>
                    <div><p></p></div>
                    <div><p></p></div>
```

#### 2.2.8 自增符号($)

```css
ul>li.item$*3    →  <ul>
                       <li class="item1"></li>
                       <li class="item2"></li>
                       <li class="item3"></li>
                   </ul>

div.box$$$*3     →  <div class="box001"></div>
                    <div class="box002"></div>
                    <div class="box003"></div>
```

#### 2.2.9 文本内容({})

```css
p{文本内容}       →  <p>文本内容</p>

a{点击这里}       →  <a href="">点击这里</a>
```

#### 2.2.10 复杂示例

```css
nav>ul>li*4>a{菜单$}

→  <nav>
       <ul>
           <li><a href="">菜单1</a></li>
           <li><a href="">菜单2</a></li>
           <li><a href="">菜单3</a></li>
           <li><a href="">菜单4</a></li>
       </ul>
   </nav>
```

```css
.container>(.row>(.col*3))*2

→  <div class="container">
       <div class="row">
           <div class="col"></div>
           <div class="col"></div>
           <div class="col"></div>
       </div>
       <div class="row">
           <div class="col"></div>
           <div class="col"></div>
           <div class="col"></div>
       </div>
   </div>
```

## 三、背景相关属性

### 3.1 背景颜色 background-color

**属性名**: `background-color`

**Emmet缩写**: `bgc`

**语法**:
```css
background-color: 颜色值;
```

**取值方式**:
```css
/* 1. 关键词 */
background-color: red;
background-color: blue;
background-color: transparent;  /* 透明(默认值) */

/* 2. 十六进制 */
background-color: #ff0000;
background-color: #f00;  /* 简写 */

/* 3. RGB */
background-color: rgb(255, 0, 0);

/* 4. RGBA(带透明度) */
background-color: rgba(255, 0, 0, 0.5);
```

**默认值**: `transparent` 或 `rgba(0,0,0,0)` (透明)

**重要特性**:
- ✅ 背景颜色**不影响盒子大小**
- ✅ 可以看清盒子的实际占用空间
- ✅ 常用于布局时查看盒子位置

**实际应用**:
```css
/* 布局时查看盒子位置 */
.container {
    width: 1200px;
    background-color: pink;  /* 方便查看盒子范围 */
}

/* 半透明背景 */
.modal {
    background-color: rgba(0, 0, 0, 0.5);
}

/* 渐变背景(需用background) */
.header {
    background: linear-gradient(to right, #667eea, #764ba2);
}
```

### 3.2 背景图片 background-image

**属性名**: `background-image`

**Emmet缩写**: `bgi`

**语法**:
```css
background-image: url(图片路径);
```

**路径写法**:
```css
/* 相对路径 */
background-image: url(./images/bg.jpg);
background-image: url(../images/bg.jpg);

/* 绝对路径 */
background-image: url(/images/bg.jpg);
background-image: url(https://example.com/bg.jpg);

/* 省略引号(推荐) */
background-image: url(./bg.jpg);
```

**重要特性**:
- ⚠️ 背景图片**不能撑开盒子**,必须设置宽高
- ✅ 默认**水平和垂直平铺**
- ✅ 仅起**装饰作用**
- ✅ url中的引号**可以省略**

**代码示例**:
```css
.box {
    width: 300px;      /* 必须设置宽度 */
    height: 200px;     /* 必须设置高度 */
    background-image: url(./bg.jpg);
}

/* 多背景图片 */
.header {
    background-image: 
        url(./logo.png),
        url(./bg.jpg);
}
```

**常见错误**:
```css
/* ❌ 错误:没有设置宽高,看不到背景图 */
.box {
    background-image: url(./bg.jpg);
}

/* ✅ 正确 */
.box {
    width: 500px;
    height: 300px;
    background-image: url(./bg.jpg);
}
```

### 3.3 背景平铺 background-repeat

**属性名**: `background-repeat`

**Emmet缩写**: `bgr`

**语法**:
```css
background-repeat: 平铺方式;
```

**取值详解**:

| 值 | 效果 | 说明 | 使用场景 |
|----|------|------|----------|
| `repeat` | 平铺 | 水平垂直都平铺(默认) | 纹理背景 |
| `no-repeat` | 不平铺 | 最常用 | Logo、图标 |
| `repeat-x` | 水平平铺 | 只在X轴平铺 | 横向背景 |
| `repeat-y` | 垂直平铺 | 只在Y轴平铺 | 纵向背景 |

**代码示例**:
```css
/* Logo不平铺 */
.logo {
    width: 200px;
    height: 60px;
    background-image: url(./logo.png);
    background-repeat: no-repeat;
}

/* 横向导航背景 */
.nav {
    background-image: url(./nav-bg.png);
    background-repeat: repeat-x;
}

/* 侧边栏背景 */
.sidebar {
    background-image: url(./sidebar-bg.png);
    background-repeat: repeat-y;
}

/* 纹理背景 */
body {
    background-image: url(./texture.png);
    background-repeat: repeat;  /* 可省略,默认值 */
}
```

### 3.4 背景位置 background-position

**属性名**: `background-position`

**Emmet缩写**: `bgp`

**语法**:
```css
background-position: 水平位置 垂直位置;
```

#### 3.4.1 方位名词写法

**水平方向**: `left`、`center`、`right`
**垂直方向**: `top`、`center`、`bottom`

```css
/* 九宫格位置 */
background-position: left top;      /* 左上 */
background-position: center top;    /* 中上 */
background-position: right top;     /* 右上 */

background-position: left center;   /* 左中 */
background-position: center center; /* 居中(最常用) */
background-position: right center;  /* 右中 */

background-position: left bottom;   /* 左下 */
background-position: center bottom; /* 中下 */
background-position: right bottom;  /* 右下 */

/* 简写 */
background-position: center;  /* 水平垂直都居中 */
background-position: left;    /* 左侧垂直居中 */
```

**方位名词特点**:
- ✅ 只写一个值,另一个默认`center`
- ✅ 两个值**可以颠倒**顺序
- ✅ 语义明确,易读

#### 3.4.2 精确数值写法

```css
/* 数字 + px */
background-position: 100px 50px;   /* 右移100px,下移50px */
background-position: -50px 0;      /* 左移50px */
background-position: 0 -100px;     /* 上移100px */

/* 百分比 */
background-position: 50% 50%;      /* 居中 */
background-position: 100% 100%;    /* 右下角 */
```

**数值特点**:
- 第一个值:**水平**方向(正右负左)
- 第二个值:**垂直**方向(正下负上)
- 只写一个值,第二个默认`center`
- 更精确的控制位置

#### 3.4.3 混合使用

```css
background-position: left 50px;    /* 左侧,向下50px */
background-position: center 100px; /* 水平居中,向下100px */
background-position: 100px center; /* 右移100px,垂直居中 */
background-position: right -20px;  /* 右侧,向上20px */
```

**注意**: 第一个值是水平,第二个值是垂直

#### 3.4.4 实际应用

```css
/* 精灵图定位 */
.icon-home {
    width: 20px;
    height: 20px;
    background-image: url(./sprite.png);
    background-repeat: no-repeat;
    background-position: -100px -50px;  /* 精确定位 */
}

/* Logo居中 */
.logo {
    width: 100%;
    height: 200px;
    background-image: url(./logo.png);
    background-repeat: no-repeat;
    background-position: center center;
}

/* 背景右上角 */
.banner {
    background-image: url(./decoration.png);
    background-repeat: no-repeat;
    background-position: right top;
}

/* 微调位置 */
.header {
    background-image: url(./bg.jpg);
    background-position: center -50px;  /* 向上偏移 */
}
```

### 3.5 背景复合属性 background

**属性名**: `background` (复合属性)

**Emmet缩写**: `bg`

**完整语法**:
```css
background: color image repeat position;
```

**推荐顺序**: 背景色 图片 平铺 位置

**代码示例**:
```css
/* 完整写法 */
background: pink url(./bg.jpg) no-repeat center center;

/* 等同于 */
background-color: pink;
background-image: url(./bg.jpg);
background-repeat: no-repeat;
background-position: center center;
```

**省略规则**:
```css
/* 可以按需省略 */
background: url(./bg.jpg);
background: pink url(./bg.jpg);
background: url(./bg.jpg) no-repeat;
background: url(./bg.jpg) center;
background: pink;
```

**特殊简写**:
```css
/* PC端常用:盒子大小=图片大小 */
background: url(./logo.png);

/* 等同于 */
background-image: url(./logo.png);
```

**实际应用**:
```css
/* 导航栏背景 */
.nav {
    background: #333 url(./nav-bg.png) repeat-x center top;
}

/* 英雄区背景 */
.hero {
    background: url(./hero.jpg) no-repeat center center;
    background-size: cover;  /* 单独设置 */
}

/* 卡片背景 */
.card {
    background: white url(./pattern.png) repeat;
}
```

**层叠问题** ⚠️:

```css
/* ❌ 错误:单独样式被覆盖 */
.box {
    background-color: red;
    background: url(./bg.jpg);  /* color被覆盖,变成透明 */
}

/* ✅ 解决方法一:单独样式写在连写后面 */
.box {
    background: url(./bg.jpg);
    background-color: red;  /* 不会被覆盖 */
}

/* ✅ 解决方法二:写在连写里面 */
.box {
    background: red url(./bg.jpg) no-repeat;
}

/* ✅ 解决方法三:使用单独属性 */
.box {
    background-image: url(./bg.jpg);
    background-color: red;
    background-repeat: no-repeat;
}
```

### 3.6 背景大小 background-size (补充)

**语法**:
```css
background-size: 宽度 高度;
```

**常用取值**:
```css
/* 像素值 */
background-size: 500px 300px;

/* 百分比 */
background-size: 100% 100%;  /* 铺满盒子 */

/* 关键词 */
background-size: cover;      /* 覆盖(常用) */
background-size: contain;    /* 包含 */
```

**cover vs contain**:
- `cover`: 图片完全**覆盖**盒子,可能裁剪
- `contain`: 图片完全**包含**在盒子内,可能留白

```css
/* 响应式背景图 */
.banner {
    background: url(./hero.jpg) no-repeat center;
    background-size: cover;  /* 始终铺满 */
}
```

### 3.7 img标签 vs 背景图片

| 对比项 | img标签 | 背景图片 |
|--------|---------|----------|
| **性质** | HTML标签 | CSS样式 |
| **撑开盒子** | ✅ 能撑开 | ❌ 不能撑开 |
| **默认显示** | 原始尺寸 | 需设置盒子大小 |
| **语义** | 内容图片 | 装饰图片 |
| **SEO** | 有利(可加alt) | 不利 |
| **加载** | 必须加载 | 可延迟加载 |
| **使用场景** | 重要图片 | 装饰图片 |

**使用建议**:
```html
<!-- ✅ 产品图片用img -->
<img src="product.jpg" alt="iPhone 15 Pro">

<!-- ✅ 用户头像用img -->
<img src="avatar.jpg" alt="张三">

<!-- ✅ 装饰性图片用背景 -->
<div class="banner" style="background: url(banner.jpg)"></div>

<!-- ✅ 图标用背景(精灵图) -->
<i class="icon icon-home"></i>
```

**选择流程图**:
```
是否重要内容? 
├─ 是 → 使用img标签
│   └─ 产品图、头像、新闻图片
└─ 否 → 使用背景图片
    └─ 装饰图、背景纹理、图标
```

## 四、元素显示模式

### 4.1 块级元素 (block)

**定义**: 独占一行的元素

**转换方式**:
```css
display: block;
```

**显示特点**:
1. ✅ **独占一行**(一行只能显示一个)
2. ✅ **宽度默认**是父元素的100%
3. ✅ **高度默认**由内容撑开
4. ✅ **可以设置**宽高

**代表标签**:
```html
<!-- 布局标签 -->
<div></div>
<header></header>
<nav></nav>
<section></section>
<article></article>
<aside></aside>
<footer></footer>

<!-- 标题标签 -->
<h1></h1> ~ <h6></h6>

<!-- 段落标签 -->
<p></p>

<!-- 列表标签 -->
<ul></ul>
<ol></ol>
<li></li>
<dl></dl>
<dt></dt>
<dd></dd>

<!-- 表单标签 -->
<form></form>
```

**代码示例**:
```html
<div class="box1">盒子1(独占一行)</div>
<div class="box2">盒子2(独占一行)</div>
<p>段落(独占一行)</p>
```

```css
.box1 {
    width: 200px;      /* 可设置宽度 */
    height: 100px;     /* 可设置高度 */
    background-color: pink;
}

.box2 {
    /* 不设置宽度,默认100% */
    height: 100px;
    background-color: skyblue;
}
```

**实际应用**:
```css
/* 页面布局 */
.container {
    width: 1200px;
    margin: 0 auto;
}

.header {
    width: 100%;
    height: 80px;
}

.content {
    width: 100%;
    min-height: 500px;
}
```

### 4.2 行内元素 (inline)

**定义**: 一行可以显示多个的元素

**转换方式**:
```css
display: inline;
```

**显示特点**:
1. ✅ **一行可以显示多个**
2. ✅ **宽高默认**由内容撑开
3. ❌ **不可以设置**宽高
4. ⚠️ 水平方向可设置`margin`和`padding`
5. ⚠️ 垂直方向`margin`和`padding`不生效

**代表标签**:
```html
<!-- 文本标签 -->
<span></span>
<a></a>

<!-- 格式化标签 -->
<strong></strong>
<b></b>
<em></em>
<i></i>
<ins></ins>
<u></u>
<del></del>
<s></s>
```

**代码示例**:
```html
<span>我</span>
<span>们</span>
<span>在</span>
<span>同一行</span>
```

```css
span {
    width: 100px;    /* ❌ 无效 */
    height: 50px;    /* ❌ 无效 */
    margin: 10px;    /* ⚠️ 只有左右生效 */
    padding: 10px;   /* ⚠️ 只有左右生效 */
}
```

**实际应用**:
```css
/* 文本强调 */
.text {
    color: #333;
}

.text strong {
    color: red;
}

/* 链接样式 */
a {
    color: #007bff;
    text-decoration: none;
}
```

**注意事项**:
```css
/* ❌ 错误:行内元素设置宽高无效 */
span {
    width: 200px;
    height: 100px;
}

/* ✅ 解决:转为块级或行内块 */
span {
    display: inline-block;  /* 或 block */
    width: 200px;
    height: 100px;
}
```

### 4.3 行内块元素 (inline-block)

**定义**: 结合行内和块级特点的元素

**转换方式**:
```css
display: inline-block;
```

**显示特点**:
1. ✅ **一行可以显示多个**
2. ✅ **可以设置宽高**
3. ⚠️ 默认宽高由内容撑开
4. ⚠️ 元素之间有间隙(空白符)

**代表标签**:
```html
<!-- 表单元素 -->
<input>
<textarea></textarea>
<button></button>
<select></select>

<!-- 特殊元素 -->
<img>  <!-- Chrome显示inline,但有行内块特性 -->
```

**代码示例**:
```html
<input type="text">
<input type="text">
<button>提交</button>
```

```css
input {
    width: 200px;      /* ✅ 可设置 */
    height: 40px;      /* ✅ 可设置 */
}

button {
    width: 80px;
    height: 40px;
}
```

**实际应用**:
```css
/* 导航按钮 */
.nav a {
    display: inline-block;
    width: 100px;
    height: 50px;
    text-align: center;
    line-height: 50px;
}

/* 分页按钮 */
.pagination a {
    display: inline-block;
    width: 40px;
    height: 40px;
    line-height: 40px;
    text-align: center;
    border: 1px solid #ddd;
}
```

**空白符问题**:
```html
<div class="box1">盒子1</div>
<div class="box2">盒子2</div>
<!-- 之间有空白符 -->
```

```css
/* 解决方法一:父元素设置字号为0 */
.parent {
    font-size: 0;
}

.parent .box1,
.parent .box2 {
    font-size: 14px;  /* 重置字号 */
}

/* 解决方法二:浮动 */
.box1,
.box2 {
    float: left;
}

/* 解决方法三:flex布局 */
.parent {
    display: flex;
}
```

### 4.4 三种显示模式对比

| 特性 | 块级元素 | 行内元素 | 行内块元素 |
|------|----------|----------|------------|
| **独占一行** | ✅ 是 | ❌ 否 | ❌ 否 |
| **可设置宽高** | ✅ 可以 | ❌ 不可以 | ✅ 可以 |
| **默认宽度** | 父元素100% | 内容撑开 | 内容撑开 |
| **默认高度** | 内容撑开 | 内容撑开 | 内容撑开 |
| **垂直margin** | ✅ 生效 | ❌ 不生效 | ✅ 生效 |
| **垂直padding** | ✅ 生效 | ⚠️ 不推荐 | ✅ 生效 |
| **代表标签** | div、p、h | span、a | input、img |
| **display值** | block | inline | inline-block |

### 4.5 显示模式转换

**语法**:
```css
display: block | inline | inline-block;
```

**转换对比**:

| 属性值 | 效果 | 使用场景 |
|--------|------|----------|
| `block` | 转为块级 | a标签转块级,扩大点击区域 |
| `inline` | 转为行内 | div转行内,一行显示多个 |
| `inline-block` | 转为行内块 | div转行内块,既一行显示又可设宽高 |

#### 4.5.1 转为块级元素

```css
/* 场景一:a标签扩大点击区域 */
.nav a {
    display: block;
    width: 100px;
    height: 50px;
    text-align: center;
    line-height: 50px;
}

/* 场景二:span转块级 */
span {
    display: block;
    width: 200px;
    height: 100px;
}
```

#### 4.5.2 转为行内块元素

```css
/* 场景一:div一行显示多个且可设宽高 */
.box {
    display: inline-block;
    width: 100px;
    height: 100px;
}

/* 场景二:实现水平导航 */
.menu li {
    display: inline-block;
}

.menu a {
    display: inline-block;
    padding: 10px 20px;
}
```

#### 4.5.3 转为行内元素

```css
/* 场景:div一行显示多个(不需要设宽高) */
div {
    display: inline;
}
```

**实际应用**:
```css
/* 按钮组 */
.btn-group button {
    display: inline-block;
    width: 80px;
    height: 40px;
    margin-right: 10px;
}

/* 面包屑导航 */
.breadcrumb a {
    display: inline-block;
    padding: 5px 10px;
}

/* 标签云 */
.tag {
    display: inline-block;
    padding: 5px 15px;
    margin: 5px;
    background-color: #f5f5f5;
    border-radius: 3px;
}
```

### 4.6 HTML嵌套规范

#### 4.6.1 块级元素嵌套规则

```html
<!-- ✅ 正确:div可以嵌套任意元素 -->
<div>
    <p>段落</p>
    <div>盒子</div>
    <span>文字</span>
    <img src="img.jpg">
    <a href="#">链接</a>
</div>

<!-- ❌ 错误:p标签不能嵌套块级元素 -->
<p>
    <div>错误!</div>
    <h1>错误!</h1>
    <p>错误!</p>
    <ul>错误!</ul>
</p>

<!-- ❌ 错误:h标签不建议嵌套块级元素 -->
<h1>
    <div>不推荐!</div>
</h1>
```

**规则总结**:
- ✅ `div`可以嵌套任意元素
- ❌ `p`不能嵌套`div`、`p`、`h`、`ul`等块级元素
- ❌ `h1~h6`不建议嵌套块级元素
- ❌ `dt`不能嵌套块级元素

#### 4.6.2 a标签嵌套规则

```html
<!-- ✅ 正确:a可以嵌套任意元素(除了a) -->
<a href="#">
    <div class="card">
        <img src="product.jpg">
        <h3>商品标题</h3>
        <p>商品描述</p>
    </div>
</a>

<!-- ❌ 错误:a不能嵌套a -->
<a href="#">
    外层链接
    <a href="#">内层链接(错误!)</a>
</a>
```

**规则总结**:
- ✅ `a`可以嵌套除`a`外的任意元素
- ❌ `a`不能嵌套`a`

#### 4.6.3 行内元素嵌套规则

```html
<!-- ✅ 正确:行内元素嵌套行内元素 -->
<span>
    <strong>粗体</strong>
    <em>斜体</em>
</span>

<!-- ❌ 错误:行内元素不能嵌套块级元素 -->
<span>
    <div>错误!</div>
    <p>错误!</p>
</span>

<!-- ⚠️ 特殊:a标签例外 -->
<a href="#">
    <div>a标签可以嵌套块级(特殊)</div>
</a>
```

**规则总结**:
- ✅ 行内元素可以嵌套行内元素
- ❌ 行内元素不能嵌套块级元素
- ⚠️ `a`标签是特例,可以嵌套块级元素

#### 4.6.4 嵌套规范速记

```
1. div > 任意元素  ✅
2. p > 只能嵌套行内元素  ⚠️
3. h > 只能嵌套行内元素  ⚠️
4. a > 任意元素(除了a)  ✅
5. span > 只能嵌套行内元素  ⚠️
```

### 4.7 居中方法总结

#### 4.7.1 文本/行内元素/行内块元素居中

```css
/* 方法:给父元素设置 */
.parent {
    text-align: center;
}
```

**适用于**:
- ✅ 纯文本
- ✅ `span`、`a`等行内元素
- ✅ `img`、`input`等行内块元素

**代码示例**:
```html
<div class="box">
    <span>居中文字</span>
    <img src="logo.png">
    <input type="text">
</div>
```

```css
.box {
    text-align: center;  /* 给父元素设置 */
}
```

#### 4.7.2 块级元素水平居中

```css
/* 方法:给自己设置 */
.box {
    width: 800px;     /* 必须有固定宽度 */
    margin: 0 auto;   /* 给自己设置 */
}
```

**要求**:
- ⚠️ 必须有**固定宽度**
- ⚠️ 给**元素自己**设置
- ⚠️ 只能**水平居中**

**代码示例**:
```css
.container {
    width: 1200px;
    margin: 0 auto;
}

.card {
    width: 300px;
    margin: 20px auto;  /* 上下20px,左右auto */
}
```

#### 4.7.3 单行文本垂直居中

```css
div {
    height: 100px;
    line-height: 100px;  /* 行高 = 盒子高度 */
}
```

**原理**: 行高等于盒子高度时,文字垂直居中

**代码示例**:
```html
<div class="box">单行文字垂直居中</div>
```

```css
.box {
    width: 200px;
    height: 100px;
    line-height: 100px;  /* 关键 */
    text-align: center;  /* 水平居中 */
}
```

#### 4.7.4 居中方法速查表

| 居中类型 | 方法 | 设置对象 | 要求 |
|----------|------|----------|------|
| 文本/行内/行内块 | `text-align: center` | 父元素 | 无 |
| 块级元素水平 | `margin: 0 auto` | 自己 | 有固定宽度 |
| 单行文本垂直 | `line-height = height` | 自己 | 单行文本 |

## 五、CSS三大特性

### 5.1 继承性

**定义**: 子元素默认继承父元素的某些样式(子承父业)

**核心概念**:
```
父元素样式 → 自动传递 → 子元素
```

#### 5.1.1 可继承的属性

**文字控制属性都可以继承**:

| 类别 | 属性 |
|------|------|
| **颜色** | `color` |
| **字体** | `font-size`、`font-weight`、`font-style`、`font-family`、`font` |
| **文本** | `text-align`、`text-indent`、`line-height` |
| **列表** | `list-style` |
| **可见性** | `visibility` |
| **光标** | `cursor` |

**代码示例**:
```html
<div class="parent">
    <p>我会继承父元素的样式</p>
    <span>我也会继承</span>
    <div>
        <a href="#">我还会继承</a>
    </div>
</div>
```

```css
.parent {
    color: red;
    font-size: 16px;
    font-family: "微软雅黑";
    line-height: 1.5;
}

/* 所有子元素都会继承这些样式 */
```

#### 5.1.2 不可继承的属性

**盒模型相关**:
- ❌ `width`、`height`
- ❌ `margin`、`padding`
- ❌ `border`

**背景相关**:
- ❌ `background`系列

**定位相关**:
- ❌ `position`、`top`、`left`、`right`、`bottom`

**显示相关**:
- ❌ `display`
- ❌ `float`

**代码验证**:
```html
<div class="parent">
    <div class="child">子元素</div>
</div>
```

```css
.parent {
    width: 500px;        /* ❌ 不继承 */
    height: 300px;       /* ❌ 不继承 */
    margin: 20px;        /* ❌ 不继承 */
    background: pink;    /* ❌ 不继承 */
}

.child {
    /* 子元素不会继承宽高、边距、背景 */
}
```

#### 5.1.3 继承的实际应用

**应用一: 统一网页字体**
```css
body {
    font: 14px/1.5 "Microsoft YaHei", Arial, sans-serif;
    color: #333;
}

/* 所有元素都会继承这些样式 */
```

**应用二: 去除列表样式**
```css
ul {
    list-style: none;
}

/* 所有li都会继承,去除小圆点 */
```

**应用三: 统一链接颜色**
```css
.article {
    color: #333;
}

.article a {
    color: inherit;  /* 继承父元素颜色 */
}
```

#### 5.1.4 继承失效的情况

**情况一: a标签的color**
```html
<div style="color: red;">
    <a href="#">我不会变红</a>
</div>
```

**原因**: 浏览器默认样式优先级更高

**解决**:
```css
a {
    color: inherit;  /* 强制继承 */
}
```

**情况二: h标签的font-size**
```html
<div style="font-size: 20px;">
    <h1>我不会是20px</h1>
</div>
```

**原因**: 浏览器给h标签设置了默认字号

**解决**:
```css
h1 {
    font-size: inherit;  /* 强制继承 */
}
```

#### 5.1.5 判断是否可继承

**方法一: 记忆规律**
```
文字控制属性 → 可继承 ✅
盒模型属性 → 不可继承 ❌
```

**方法二: Chrome调试工具**
```
Elements → Styles → Inherited from xxx
```

如果看到`Inherited from`,说明该属性可继承

### 5.2 层叠性

**定义**:
1. **不同属性** → 层叠**叠加** → 共同作用
2. **相同属性** → 层叠**覆盖** → 后面生效

#### 5.2.1 不同属性叠加

```css
div {
    color: red;
    font-size: 20px;
    background-color: pink;
}

/* 三个属性都会生效,共同作用在div上 */
```

**原理**: 不同属性不冲突,可以共存

#### 5.2.2 相同属性覆盖

```css
div {
    color: red;
    font-size: 20px;
    color: blue;  /* blue生效,覆盖red */
}

/* 最终:color为blue,font-size为20px */
```

**原理**: 相同属性冲突,后面的覆盖前面的

#### 5.2.3 实际应用

**场景一: 公共样式+特殊样式**
```css
/* 公共样式 */
.btn {
    padding: 10px 20px;
    border: none;
    cursor: pointer;
}

/* 特殊样式 */
.btn {
    background-color: #007bff;  /* 叠加 */
    color: white;               /* 叠加 */
}
```

**场景二: 样式覆盖**
```css
/* 默认样式 */
p {
    color: #333;
}

/* 特殊情况覆盖 */
p {
    color: red;  /* 覆盖前面的 */
}
```

#### 5.2.4 注意事项

**前提条件**:
```
只有当选择器优先级相同时,
才通过层叠性(书写顺序)判断结果
```

**示例**:
```css
/* 优先级不同,不看层叠性 */
div {
    color: red;
}

.box {
    color: blue;  /* 类选择器优先级高,blue生效 */
}
```

```css
/* 优先级相同,看层叠性 */
.box {
    color: red;
}

.box {
    color: blue;  /* 后面的覆盖前面的,blue生效 */
}
```

### 5.3 优先级 (初步认识)

**基础优先级**:
```
继承/通配符 < 标签选择器 < 类选择器 < id选择器 < 行内样式 < !important
```

**权重值**:
```
继承/通配符:  0
标签选择器:   1
类选择器:    10
id选择器:   100
行内样式:  1000
!important: ∞ (无穷大)
```

**代码示例**:
```html
<div class="box" id="container" style="color: yellow;">文字</div>
```

```css
* {
    color: gray;     /* 0 */
}

div {
    color: red;      /* 1 */
}

.box {
    color: blue;     /* 10 */
}

#container {
    color: green;    /* 100 */
}

/* 行内样式: color: yellow; (1000) */

div {
    color: pink !important;  /* ∞ 最高优先级 */
}

/* 最终生效: pink (!important最高) */
```

**注意**:
- ⚠️ `!important`优先级最高,但**不推荐使用**
- ⚠️ 权重可以**叠加计算**(后续详细学习)

| 选择器 | 符号 | 语法 | 作用 | 使用频率 |
|--------|------|------|------|----------|

#### 问题5: 层叠覆盖问题
