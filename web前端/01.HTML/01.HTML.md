# HTML学习笔记

## 一、基础认知

### 1.1 网页基础
- **网页组成**:文字、图片、音频、视频、超链接
- **运行平台**:浏览器(负责解析和渲染代码)

### 1.2 五大浏览器
- IE浏览器
- 火狐浏览器(Firefox)
- **谷歌浏览器(Chrome)** ← 推荐使用
- Safari浏览器
- 欧朋浏览器(Opera)

**渲染引擎**:不同浏览器内核不同,解析效果存在差异

### 1.3 Web标准
为了统一不同浏览器的显示效果,Web标准分为三个构成:

| 构成 | 语言 | 说明 |
|------|------|------|
| 结构 | HTML | 页面元素和内容 |
| 表现 | CSS | 网页样式(颜色、大小等) |
| 行为 | JavaScript | 页面交互和动态效果 |

### 1.4 HTML骨架结构
```html
<!DOCTYPE html>
<html>
  <head>
    <title>网页标题</title>
  </head>
  <body>
    网页主体内容
  </body>
</html>
```

- `<html>`:网页整体
- `<head>`:网页头部
- `<title>`:网页标题
- `<body>`:网页身体

### 1.5 VS Code快捷键
- 快速生成骨架:`!` + `Tab`
- 保存文件:`Ctrl` + `S`
- 注释:`Ctrl` + `/`
- 浏览器预览:`Alt` + `B`
- 复制整行:`Ctrl` + `C`
- 删除整行:`Ctrl` + `X`

## 二、HTML语法规范

### 2.1 标签构成
**双标签**:开始标签 + 结束标签
```html
<p>内容</p>
<h1>标题</h1>
```

**单标签**:自成一体
```html
<br>
<hr>
```

### 2.2 标签属性
```html
<标签名 属性名="属性值">内容</标签名>
```
- 属性写在开始标签中
- 标签名与属性、属性与属性之间用空格隔开
- 可同时存在多个属性

### 2.3 标签关系
- **父子关系(嵌套)**:标签内部包含标签
- **兄弟关系(并列)**:标签之间平级

## 五、媒体标签

### 5.1 图片标签
```html
<img src="图片路径" alt="替换文本" title="提示文本" width="宽度" height="高度">
```

**常用属性**:
- `src`:图片路径(必需)
- `alt`:图片加载失败时显示
- `title`:鼠标悬停时显示
- `width/height`:只设置一个会等比缩放

### 5.2 路径

#### 相对路径(常用)
- **同级目录**:`目标文件.jpg` 或 `./目标文件.jpg`
- **下级目录**:`文件夹名/目标文件.jpg`
- **上级目录**:`../目标文件.jpg`

#### 绝对路径(了解)
- 盘符开头:`D:\images\1.jpg`
- 完整网址:`https://example.com/image.jpg`

### 5.3 音频标签
```html
<audio src="音频路径" controls autoplay loop></audio>
```
**属性**:
- `controls`:显示控件
- `autoplay`:自动播放
- `loop`:循环播放

支持格式:MP3、Wav、Ogg

### 5.4 视频标签
```html
<video src="视频路径" controls autoplay muted loop></video>
```
**属性**:同音频,另有`muted`(静音,配合autoplay使用)

支持格式:MP4、WebM、Ogg

## 六、链接标签

```html
<a href="跳转地址" target="_self">链接文字</a>
```

**常用属性**:
- `href`:跳转地址
    - 外部链接:`https://www.baidu.com`
    - 内部链接:`./页面.html`
    - 空链接:`#`(占位用)
- `target`:打开方式
    - `_self`:当前窗口(默认)
    - `_blank`:新窗口

**显示特点**:
- 文字有下划线
- 未访问:蓝色
- 已访问:紫色

## 七、列表标签

### 7.1 无序列表(最常用)
```html
<ul>
  <li>列表项1</li>
  <li>列表项2</li>
</ul>
```
- `ul`中只能包含`li`
- `li`中可以包含任意内容

### 7.2 有序列表
```html
<ol>
  <li>列表项1</li>
  <li>列表项2</li>
</ol>
```

### 7.3 自定义列表(底部导航常用)
```html
<dl>
  <dt>主题</dt>
  <dd>内容1</dd>
  <dd>内容2</dd>
</dl>
```
- `dl`中只能包含`dt`/`dd`
- `dd`默认有缩进效果

## 八、表格标签

### 8.1 基本结构
```html
<table border="1" width="500" height="300">
  <caption>表格标题</caption>
  <thead>
    <tr>
      <th>表头1</th>
      <th>表头2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>内容1</td>
      <td>内容2</td>
    </tr>
  </tbody>
</table>
```

**嵌套关系**:`table` > `tr` > `td`

### 8.2 常用属性
- `border`:边框宽度
- `width/height`:表格宽高
- `cellspacing`:单元格间距(实际开发用CSS)

### 8.3 特殊标签
- `<caption>`:表格大标题(写在table内部)
- `<th>`:表头单元格(替换td,文字加粗居中)
- `<thead>`:表格头部
- `<tbody>`:表格主体
- `<tfoot>`:表格底部

### 8.4 合并单元格
**步骤**:
1. 明确合并哪几个单元格
2. **左上原则**:保留最左或最上的,删除其他
3. 给保留的单元格添加属性:
    - `rowspan="数字"`:跨行合并(垂直)
    - `colspan="数字"`:跨列合并(水平)

⚠️ 不能跨结构标签(thead/tbody/tfoot)合并

## 九、表单标签

### 9.1 input系列

#### 文本框
```html
<input type="text" placeholder="提示文字" name="username" value="默认值">
```

#### 密码框
```html
<input type="password" placeholder="请输入密码">
```

#### 单选框
```html
<input type="radio" name="gender" value="1" checked> 男
<input type="radio" name="gender" value="2"> 女
```
- 同组单选框`name`必须相同
- `checked`:默认选中

#### 复选框
```html
<input type="checkbox" name="hobby" value="1"> 篮球
<input type="checkbox" name="hobby" value="2" checked> 足球
```

#### 文件选择
```html
<input type="file" multiple>
```
- `multiple`:多文件选择

#### 按钮
```html
<input type="submit" value="提交">
<input type="reset" value="重置">
<input type="button" value="普通按钮">
```

### 9.2 button按钮
```html
<button type="submit">提交</button>
<button type="reset">重置</button>
<button type="button">普通按钮</button>
```
- 双标签,可包裹图片等内容
- 默认是提交按钮

### 9.3 下拉菜单
```html
<select>
  <option>北京</option>
  <option selected>上海</option>
  <option>广州</option>
</select>
```
- `selected`:默认选中

### 9.4 文本域
```html
<textarea cols="30" rows="10"></textarea>
```
- 可输入多行文本
- 右下角可拖拽调整大小

### 9.5 label标签
增强用户体验,点击文字即可选中表单

**方法一**:
```html
<input type="radio" id="nan">
<label for="nan">男</label>
```

**方法二**:
```html
<label>
  <input type="radio"> 男
</label>
```

### 9.6 表单域
```html
<form action="提交地址" method="post">
  <!-- 表单控件 -->
</form>
```
- 提交按钮需配合form使用

## 十、语义化标签

### 10.1 无语义布局标签
- `<div>`:独占一行
- `<span>`:一行可显示多个

### 10.2 有语义布局标签(HTML5)
```html
<header>头部</header>
<nav>导航</nav>
<section>区块</section>
<article>文章</article>
<aside>侧边栏</aside>
<footer>底部</footer>
```
- 显示特点与div相同
- 语义更清晰,利于SEO

## 十一、字符实体

HTML中多个空格/换行会合并成一个空格,需用字符实体显示特殊符号:

| 显示 | 字符实体 |
|---|----------|
| 空格 | `&nbsp;` |
| < | `&lt;` |
| > | `&gt;` |

**结构**:`&实体名;`

---

## 十二、开发规范

1. **标签语义化**:根据内容选择合适标签
2. **属性顺序**:无固定要求,但建议常用属性前置
3. **代码缩进**:保持良好缩进,提高可读性
4. **注释规范**:重要代码段添加注释
5. **文件命名**:小写字母,用`-`连接,如`user-info.html`
