
# Element-Plus

## 一、Element Plus 导入

### 1. 安装

使用你喜欢的包管理器进行安装：

```bash
# NPM
npm install element-plus --save

# Yarn
yarn add element-plus

# pnpm
pnpm install element-plus
```

### 2. 引入方式

#### A. 自动导入（推荐）
这是最推荐的方式，可以自动按需引入组件和样式，减少项目体积。

1. **安装插件**：
```bash
npm install -D unplugin-vue-components unplugin-auto-import
```

2. **配置 Vite**（`vite.config.ts`）：
```typescript
import { defineConfig } from 'vite'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    // ...
    AutoImport({
      resolvers: [ElementPlusResolver()],
    }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
})
```

#### B. 完整引入
如果不介意打包体积，或者在开发初期为了方便，可以选择完整引入。

在 `main.ts` 或 `main.js` 中配置：
```typescript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus)
app.mount('#app')
```

#### C. 手动引入
需要配合 `unplugin-element-plus` 来实现样式的自动导入。

1. **安装插件**：`npm install -D unplugin-element-plus`

2. **在组件中使用**：
```vue
<template>
  <el-button>按钮</el-button>
</template>

<script setup>
import { ElButton } from 'element-plus'
</script>
```

### 3. 全局配置
在引入 Element Plus 时，可以传入一个全局配置对象（如设置组件默认尺寸和弹窗层级）。
```typescript
app.use(ElementPlus, {
  size: 'small',
  zIndex: 3000
})
```

### 4. 国际化（中文配置）
Element Plus 默认使用英语。如需使用中文，需进行以下配置：

**完整引入时：**
```typescript
import ElementPlus from 'element-plus'
import zhCn from 'element-plus/es/locale/lang/zh-cn'

app.use(ElementPlus, {
  locale: zhCn,
})
```

**组件方式引入时（推荐）：**
使用 `ElConfigProvider` 组件包裹根节点：
```vue
<template>
  <el-config-provider :locale="zhCn">
    <app />
  </el-config-provider>
</template>

<script setup>
import { ElConfigProvider } from 'element-plus'
import zhCn from 'element-plus/es/locale/lang/zh-cn'
</script>
```

### 5. 注意事项

1. **浏览器兼容性**：Element Plus 不再支持 IE 浏览器（因为 Vue 3 不支持）。
2. **Sass 版本**：如果使用 Sass，建议确保版本符合要求（>= 1.79.0），避免废弃 API 的警告。
3. **IDE 支持**：如果使用 VSCode，建议安装 **Vue Language Features (Volar)** 插件以获得更好的开发体验。
